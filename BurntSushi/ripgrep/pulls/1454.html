globset: Add GlobMatcher::glob

#1454 opened by @LPGhatguy on 2020-01-03 23:41
@LPGhatguy

The GlobMatcher type holds onto the Glob object that was used to create it. This PR exposes a reference to it through a getter.

My main use is that I have a project that uses globset by wrapping the GlobMatcher type in a type that I can implement Serde's Serialize and Deserialize on. In order to get access to the original pattern, I currently need to redundantly keep my own copy of the Glob struct to get access to its string pattern.

This API would enable me to only need to keep around GlobMatcher.
