Why The PR?

Github has a really nice feature where you can copy markdown code blocks to your clipboard. The README.md for this repo also has all the common commands specified to install this package depending on your environment.

However, the commands include a prompt character. So when you copy and paste the commands they do not work without modification.

This PR aims to remedy this!
