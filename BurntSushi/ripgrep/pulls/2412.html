@vincentbockaert

The default filter for terraform only checks for *.tf files, but there are quite few other terraform filetypes.

The explanation for all of them can be found below: (including a link to the documentation from Hashicorp at the time of writing)

  • *.tf.json & *.tfvars.json is to capture the files written in JSON-based variant of the Terraform language
  • *.tfvars is used to supply variables
  • .terraform.lock.hcl is used as a Dependency lock file
  • terraform.rc & .terraformrc, *.tfrc
