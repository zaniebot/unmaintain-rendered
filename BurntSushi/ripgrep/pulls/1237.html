searcher: add option to disable BOM sniffing

#1237 opened by @BurntSushi on 2019-04-06 13:20
@BurntSushi

This commit adds a new encoding feature where the -E/--encoding flag will now accept a value of 'none'. When given this value, all encoding related machinery is disabled and ripgrep will search the raw bytes of the file, including the BOM if it's present.
