I do not have an aarch64 Windows machine to test with at the moment, but I can confirm the architecture of rg.exe is arm64 per Git Bash's file, and it appears the test suite succeeded in CI.

This PR only adds MSVC support for aarch64, not GNU. Looking at the available toolchains, it seems the only GNU toolchain for Windows aarch64 is the -gnullvm version, which ripgrep is not using at the moment.
