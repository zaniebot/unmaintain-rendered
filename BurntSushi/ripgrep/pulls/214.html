I remembered this article "Rendering differences in prose documents". So I add filetypes refered in it. These filetypes can be previewed in Github.

Filetypes

  • Textile : a lightweight markup language that uses a text formatting syntax to convert plain text into structured HTML markup.
  • Org(Org Mode) : for keeping notes, maintaining TODO lists, planning projects, and authoring documents with a fast and effective plain-text system
  • Rdoc : Documentation from Ruby Source Files
  • MediaWiki : wiki markup used by MediaWiki
  • Pod : "Plain Old Documentation", is a simple-to-use markup language used for writing documentation for Perl, Perl programs, and Perl modules.
  • Creole : A common wiki markup for the wiki ohana
