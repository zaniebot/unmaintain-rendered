Comment by @BurntSushi on 2017-07-18 16:10

Problems are update-synchonisation

I don't think this can be characterized as a problem. If anything, it's a trade off. Bundling is useful because it means there will never be a version mismatch between what the tool expects and what ripgrep actually does. For example, if a new release of ripgrep changes some behavior that the other tool depends on, then the tool might break. If ripgrep is bundled, then behavior always remains the same.
