Comment by @BurntSushi on 2020-08-26 14:58

This is correct. Specifying an override creates a whitelist, which means that all paths yielded must be in that whitelist. The behavior you're asking for doesn't really make sense. e.g., rg foo -g '*.rs' for example would wind up searching everything anyway. And that's the use case that Overrides is intended for.
