Add option '-l' to only print out the file names from the matched result

#97 opened by @nextlogic-ono on 2016-09-26 03:33
@nextlogic-ono

For grep and the likes including ack, they have an option that only prints the file names from the matched results.

grep -l, --files-with-matches Suppress normal output; instead print the name of each input file from which output would normally have been printed. The scanning will stop on the first match.

ack -l, --files-with-matches Only print the filenames of matching files, instead of the matching text.

This is useful for example when opening the target files in vim directly from the result, as in;

vim -p grep -l foo *
