The sort option --sort path sort the result by their file path. But it doesn't respect the case sensitivity of the file system. The result is sorted case-sensitively.

AFAIK, the correct way to implement this is to use an implementation of the Unicode collation algorithm. A high quality implementation of this does not exist in Rust yet. When it does, we can revisit this. Until then, I'm going to unfortunately mark this as wontfix. Sorry.
