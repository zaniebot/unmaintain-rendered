Comment by @BurntSushi on 2020-05-09 16:51

Since I've been linking people to this issue when they ask for a chat room, I figured I'd add a bit more context and some updates.

Firstly, I no longer hang around on IRC. The only chat rooms I visit these days are the Rust Discord server and the rust-lang Zulip server. If you ping me on either of those places, I should see it.

Secondly, the problem with an officially endorsed chat room is that they require moderation. I am on the Rust moderation team, but I cannot spend my days sitting in a chat room moderating it.

As a compromise, I reached out to GitHub and specifically requested access to their Discussions beta program. You are now encouraged to post questions---any kind of question---in ripgrep's discussion board: https://github.com/BurntSushi/ripgrep/discussions
