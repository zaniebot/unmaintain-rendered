Describe your question, feature request, or bug.

ripgrep supports the type java, but it only includes *.java and *.jsp. However, properties for java files are usually stored in *.properties files. (https://beyondgrep.com/)[ack-grep] also supports a --java option, which includes this. ripgrep currently has no support for *.properties files at all. This should be changed and I think included in the java type. See the bottom of https://beyondgrep.com/documentation/ for the type supported by ack-grep.
