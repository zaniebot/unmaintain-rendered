### Describe your use case

Other CLIs (ocaml for example) use `-?` in addition to `-h`:

```
arty.exe c -?
Compile into verilog

  arty.exe compile 

=== flags ===

  [-output Output]           . file
  [-help], -?                . print this help text and exit
```

### Describe the solution you'd like

Alias `-h` with `-?`
