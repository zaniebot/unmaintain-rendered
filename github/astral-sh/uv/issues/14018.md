# Install of pydevd_pycharm doesn't work

---

_Issue opened by @david-banon-tecnativa on 2025-06-13 10:58_

### Summary

When installin pydevd_pycharm using uv, the package doesn't work and throws an issue. However, installing with regular pip does not.

How to replicate:
```bash
mkdir /tmp/pydev_test
python -m venv /tmp/pydev_test/.pip
python -m venv /tmp/pydev_test/.uv
```
Place this file on /tmp/pydev_test/
```python
import pydevd_pycharm
pydevd_pycharm.settrace('localhost', port=12345, stdoutToServer=True, stderrToServer=True)
test = "test"
print(test)
```
For pip:
```
source /tmp/pydev_test/.pip/bin/activate
pip install pydevd pydevd_pycharm
python main.py
```
For UV:
```
source /tmp/pydev_test/.uv/bin/activate
uv pip install pydevd pydevd_pycharm
python main.py
```

If a pydev debugger is running, pip version successfully connects to debugger.
but the uv version throws this error:
