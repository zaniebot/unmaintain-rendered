# `uv python install` tries to install x86_64 binary on arm64 Windows

---

_Issue opened by @saschanaz on 2025-05-08 06:44_

### Summary

```
> uv python install 3.13
cpython-3.13.3-windows-x86_64-none (download) ------------------------------ 9.95 MiB/20.15 MiB   
```

It should get arm64 binary instead.

### Platform

Windows 11 arm64

### Version

uv v0.7.3

### Python version

3.12.10

---

_Comment by @zanieb on 2025-05-08 14:13_

We don't support installing arm Windows Python versions yet, and even once we do, support in Python packages is very limited and we will likely not install it by default.

x86-64 is transparently emulated on arm64, so it's fine to use.
