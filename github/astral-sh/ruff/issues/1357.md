# Option to enable all lints

---

_Issue opened by @bluetech on 2022-12-24 09:22_

My preferred usage is to enable all lints, then `extend-ignore` the ones I don't want with a comment. (This strategy only makes sense if using a pinned version of ruff).

Currently to enable all available lints, I list all of the prefixes in `extend-select`, but that's a bit long and won't catch any new prefixes if/when they become available.

I tried `extend-select = [""]` (empty string being a prefix of everything), but it's not accepted.

Maybe there's a way to do this that I missed?
