# formatter should not concatenate multiple strings on the same line

---

_Issue opened by @DetachHead on 2024-02-12 23:15_

# before
```py
asdf(
    "asdf"
    "asdf"
)
```
# after
## actual
```py
asdf("asdf" "asdf")
```
## expected
it should either do this:
```py
asdf("asdfasdf")
```
or just leave it as-is
```py
asdf(
    "asdf"
    "asdf"
)
```
