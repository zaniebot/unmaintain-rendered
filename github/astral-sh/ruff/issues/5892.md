# Formatter: arguments get wrong trailing comma

---

_Issue opened by @konstin on 2023-07-19 18:52_

We format
```python
def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x: y,
):
    pass

def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa = lambda x: y,
):
    pass
```
as

```python
def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x,: y,
):
    pass


def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa=lambda x: y,
):
    pass
```
(notice the trailing comma after the `x`). It needs to be investigated why we add the comma in the first place

This causes unstable formatting:

```python
def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x: x,
):
    pass
```

```
---
--- Formatted once
+++ Formatted twice
@@ -1,4 +1,5 @@
 def f(
-    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x,: x,
+    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x,
+    : x,
 ):
     pass
---

Formatted once:
---
def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x,: x,
):
    pass
---

Formatted twice:
---
def f(
    aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaa: bbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbbb = lambda x,
    : x,
):
    pass
---
```

---

I assume this instability is the same bug:

```python
f(s=lambda d,id,blob,timestamp=None,encryption_revision=1,sequester_blob=None:{},)
```

```
---
--- Formatted once
+++ Formatted twice
@@ -4,5 +4,6 @@
     blob,
     timestamp=None,
     encryption_revision=1,
-    sequester_blob=None,: {},
+    sequester_blob=None,
+    : {},
 )
---

Formatted once:
---
f(
    s=lambda d,
    id,
    blob,
    timestamp=None,
    encryption_revision=1,
    sequester_blob=None,: {},
)
---

Formatted twice:
---
f(
    s=lambda d,
    id,
    blob,
    timestamp=None,
    encryption_revision=1,
    sequester_blob=None,
    : {},
)
---
```
