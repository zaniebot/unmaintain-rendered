# [flake8-aaa](https://pypi.org/project/flake8-aaa/)

Enforces the AAA (Arrange-Act-Assert) pattern for test functions. As such it should only run for test files.

### Error Codes
* [ ] `AAA01` no Act block found in test
* [ ] `AAA02` multiple Act blocks found in test
* [ ] `AAA03` expected 1 blank line before Act block, found none
* [ ] `AAA04` expected 1 blank line before Assert block, found none
* [ ] `AAA05` blank line in block
* [ ] `AAA06` comment in Act block
