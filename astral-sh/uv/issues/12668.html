Question

I've made this repo is to investigate how to make uv workspaces work properly with streamlits hot reload feature.

Project structure 
├── apps
│   └── streamlit-app
│       ├── main.py
│       └── pyproject.toml
├── packages
│   └── hello-world
│       ├── pyproject.toml
│       └── src
│           └── hello_world
│               ├── __init__.py
│               └── py.typed
├── pyproject.toml
└── uv.lock
The problem

If I run my streamlit app

uv run streamlit run apps/streamlit-app/main.py

then the hot reload works just fine if I make changes in apps/streamlit-app. However any changes to packages/hello-world is not picked up.

Streamlit suggests to update PYTHONPATH but that's not working together with uv workspaces AFAIK.

Note I have asked the same question in streamlit issues since I'm not sure where this belongs.

Platform

Darwin 24.3.0 arm64

Version

uv 0.6.12 (e4e03833f 2025-04-02)
