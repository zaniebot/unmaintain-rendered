Support auto backend selection for CuPy

#17136 opened by @MattTheCuber on 2025-12-15 17:04
@MattTheCuber

Summary

CuPy installation offers specific backend selection to speed up the installation process. For example, you need to use uv pip install cupy-cuda13x on a CUDA 13 system (see the docs here).

The UV documentation describes using --torch-backend=auto, it would be nice if this would also apply to CuPy.

Example

uv pip install cupy --backend=auto
