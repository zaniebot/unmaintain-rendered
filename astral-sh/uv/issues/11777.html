Create a statically linked binary for python projects

#11777 opened by @kishaningithub on 2025-02-25 10:23
@kishaningithub

Summary

If i run something like uv compile my python project should produce a statically linked cross-platform binary.

This is inspired from deno compile which creates statically linked cross-platform binaries for Javascript/Typescript.

Example

  • New command called uv compile which produces a statically linked binary just like deno compile
  • An alternative design option would be to use uv build to create binaries in addition to whl and zip which it is creating now
