#16875 opened by @ezzarghili on 2025-11-27 14:11

Question

When running uv python install graalpy, only GraalPy 3.11 is available for download. The latest GraalPy releases (such as 3.12 with Java 25.0.1) are not downloadable.

Steps to Reproduce:

  1. Run:
uv python install graalpy
  1. Observe the available versions.

Expected Behavior: The latest GraalPy versions, including 3.12 (Java 25.0.1), should be available for installation through uv.

Actual Behavior: version 3.11 is offered:
