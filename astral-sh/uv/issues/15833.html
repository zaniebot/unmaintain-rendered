Comment by @nick-voisin on 2025-09-14 08:58

Hi,

I think what you want is to do uv init --lib my_project. My uv.lock looks like the following for test-project:

`version = 1
revision = 3
requires-python = ">=3.12"

[[package]]
name = "test-project"
version = "0.1.0"
source = { editable = "." }

And imports work fine.
