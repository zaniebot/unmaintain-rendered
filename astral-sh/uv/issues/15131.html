I don't think ~/.config/uv/uv.toml is being detected. On Windows, isn't it %APPDATA%\uv\uv.toml? You would see it in the debug logs if it were being picked up.

Comment by @charliermarsh on 2025-08-08 09:21

You can use https://docs.astral.sh/uv/reference/settings/#allow-insecure-host to skip verification, though we of course don't recommend it.
