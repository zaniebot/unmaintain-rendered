Question

My use case is that I'm using docker and installing dependencies requires pyproject.toml:

COPY uv.lock pyproject.toml ./
RUN /opt/.venv/bin/uv sync

So is it possible to install dependencies only from uv.lock? From looking at the uv docs I don't think so, but I wanted to check. If not, is this something you'd be open to supporting?
