uv run seems to strip quotes from the command, and I'm yet to figure out a way around it.

Demo of the problem:

  1. Dump the following into demo.sh
#!/bin/bash 
for i; do 
   echo $i
done
  1. Run uv run bash demo.sh foo bar and uv run bash demo.sh "foo bar" and see how both treat foo and bar as two args and not one in the second case. uv run bash demo.sh ""foo bar"" also behaves exactly the same...

Version: 0.2.33
