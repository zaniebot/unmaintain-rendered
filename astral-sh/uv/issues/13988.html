Do not allow uv add --group ... --script

#13988 opened by @zanieb on 2025-06-12 13:07
@zanieb

uv will happily do

uv add --group science numpy --script main.py

to create

# /// script
# requires-python = ">=3.13"
#
# [dependency-groups]
# science = [
#     "numpy",
# ]
# ///

it doesn't like

uv sync --group science --script main.py

Originally posted by @konstin in https://github.com/astral-sh/uv/pull/13735#discussion_r2142078192

This isn't allowed by the specification.
