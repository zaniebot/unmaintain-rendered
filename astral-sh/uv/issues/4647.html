Looks like jemalloc is not supported on that operating system e.g. home-assistant/core#105768

Home-assistant provides a DISABLE_JEMALLOC option to work-around this, we might need to do similar?

It looks like they also changed the page size (i.e. JEMALLOC_SYS_WITH_LG_PAGE) which may solve the issue? home-assistant/docker-base#248 and more discussion at qdrant/qdrant#3945#issuecomment-2030026387
