Issue opened by @yshiftanlightricks on 2025-12-23 13:21

Summary

Our project can be built with uv lock only when using uv version 0.9.0 or later. We would really appreciate the ability to enforce the uv version in pyproject.toml. I understand that if someone already has an older version installed, they might not get a clear error message. However, at least for new installations or newer versions, this would be very helpful.

Comment by @sebadevo on 2025-12-23 14:41

to check the uv version you can add the following in your pyproject.toml

[tool.uv]
required-version = ">=0.9.0"
