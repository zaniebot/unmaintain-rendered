Comment by @charliermarsh on 2024-10-01 12:57

Your pyproject.toml does not define static metadata for the package. So by rule we need to ask the build backend for the metadata. Which requires invoking prepare_metadata_for_build_wheel. You can get around this by defining metadata in the pyproject.toml:

[project]
name = "pyo3_example"
version = "0.0.1"
dependencies = []

[build-system]
requires = ["maturin>=1,<2"]
build-backend = "maturin"
