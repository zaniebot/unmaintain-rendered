This is a feature request

I like uv init to create simple ad-hoc projects and often use it with the exact same options. I would love to be able to drive this via a configuration, such as a uv.toml in a "one-off apps" folder.

For consistency with other subcommands it could look like this:

# uv.toml

[init]
app = true
vcs = "none"
