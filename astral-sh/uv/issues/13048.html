Hi, Pardon my lack of Linux experience in advance...

I'm trying to get UV installed on my Linux machine to make it available to all other users.

The below command will install it on the correct dir, but fail to move files from tmp to the specified dir. (Even with sudo) curl -LsSf https://astral.sh/uv/install.sh | env UV_INSTALL_DIR="/usr/local/bin" sh

As /usr/local/bin requires extra privileges, would uv ever be able to self-update if the installation succeeds? Any idea how I could achieve this?

Comment by @konstin on 2025-04-22 09:08

The sudo needs to be after the pipe:

curl -LsSf https://astral.sh/uv/install.sh | env UV_INSTALL_DIR="/usr/local/bin" sudo sh

Comment by @adammccartney on 2025-08-13 08:37

To get this working, I needed to also tell sudo to use the specific environment:

curl -LsSf https://astral.sh/uv/install.sh | env UV_INSTALL_DIR="/usr/local/bin" sudo -E sh
