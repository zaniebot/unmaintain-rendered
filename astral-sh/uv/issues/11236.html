Decided to look into the rust code, lord bless my soul.

If the first attempt fails, a second attempt is made to check for host credentials. See keyring.rs It's a bit finicky (No ending backslash, no protocol - these are present in the absolute URI), but: IF keyring get private-sandbox.com Token yields the token, the token is successfully retrieved.

This seems like a solid workaround for my case. I do think the docs need some love on this topic.
