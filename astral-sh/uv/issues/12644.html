Comment by @charliermarsh on 2025-04-03 12:55

This is intentional. uv pip sync installs exactly the dependencies in the input file -- it does not look at transitive dependencies (and removes anything extraneous). Typically, you'd run uv pip compile, then uv pip sync on the generated file.
