Feature request: Install only packages from an optional group

#8292 opened by @chris48s on 2024-10-17 15:45
@chris48s

Sometimes it would be useful to be able to install only the packages from an optional group (but not the main or dev dependencies).

For example: When building a documentation site, or deploying an application in CI it might be useful to only install the packages relating to documentation or deployment without also installing the application's full tree of runtime and/or dev dependencies - particularly for larger projects.

Poetry does heave this feature via the --only flag e.g: poetry install --only <group>

As far as I can tell, uv sync doesn't allow this via any combination of flags.

Comment by @charliermarsh on 2024-10-30 20:18
