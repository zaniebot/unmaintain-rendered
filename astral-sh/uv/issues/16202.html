Comment by @konstin on 2025-10-09 15:34

You can find documentation in https://docs.astral.sh/uv/reference/environment/, and we're extending the uv paths documentation in https://github.com/astral-sh/uv/pull/15954. For a project, uv always uses .venv in a directory of either the workspace root, or of the project if there is not workspace. There's no command that shows this location currently.

Comment by @zanieb on 2025-10-09 15:59
