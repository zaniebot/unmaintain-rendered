`uv add` should not remove version specifiers when adding source

#5339 opened by @zanieb on 2024-07-23 15:50
Issue opened by @zanieb on 2024-07-23 15:50

e.g. the version specifier is removed from httpx in project.dependencies

$ uv init
$ uv add 'httpx>0.1.0'
$ uv add git+https://github.com/encode/httpx
$ cat pyproject.toml
[project]
name = "example"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
dependencies = [
    "httpx",
]

[tool.uv]
dev-dependencies = []

[tool.uv.sources]
httpx = { git = "https://github.com/encode/httpx" }
