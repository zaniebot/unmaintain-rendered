`XDG_BIN_HOME` common default not used

#6937 opened by @thernstig on 2024-09-02 12:44
Issue opened by @thernstig on 2024-09-02 12:44

In case someone has not set XDG_BIN_HOME, it installs into ~/.cargo/bin/ on Linux.

A common convention is however to install into $HOME/.local/bin. Example: https://gist.github.com/roalcantara/107ba66dfa3b9d023ac9329e639bc58c

The benefit is of course the same as with anything else for the XDG spec. It keeps the home dir clutter free.
