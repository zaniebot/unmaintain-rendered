Comment by @samypr100 on 2024-12-22 03:35

Looking at their setup.py I notice they leverage PEP517_BUILD_BACKEND environment variable being set to patch the include files (this variable seems to be set by pip). I think this would be set to setuptools.build_meta for this project and using_build_backend will be True in their build script. When using uv this would be False as those environment variables will not be set.

TL;DR try setting PEP517_BUILD_BACKEND=setuptools.build_meta before your commands.
