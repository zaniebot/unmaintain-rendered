Add a `uv sync --check` flag

#12338 opened by @liiight on 2025-03-20 12:29
Issue opened by @liiight on 2025-03-20 12:29

Summary

I would like to have the ability to check if an environment is synced without actually syncing it. I am aware that there is currently a --dry-run flag and I am currently using it like so:

uv sync --dry-run | grep -qo "Would make no changes"

But that feels a little awkward and flaky. A proper --check flag that would return a relevant exit code would be great.

Example

uv sync --check
