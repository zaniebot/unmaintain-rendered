Add an option to dump PEP 517 metadata

#16639 opened by @leodevian on 2025-11-07 18:06

Summary

Hello,

pypa/build added an option to bump PEP 517 metadata. I would love to be able to use this option with the uv build command.

Example

This command would display the project version, whether it is dynamic or static:

uv build --metadata | jq -r '.version'

It could be used in CI/CD jobs to verify the deployment of packages to package indexes or to remove them though API requests.

uv build --metadata > metadata.json
name=$(jq -r '.name' < metadata.json)
version=$(jq -r '.version' < metadata.json)
