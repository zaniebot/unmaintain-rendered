I noticed that the comments shows where uv install python by uv python install, and I just want to ensure whether I'm right

  1. If env var UV_PYTHON_INSTALL_DIR is set, install python under UV_PYTHON_INSTALL_DIR;
  2. If UV_PYTHON_INSTALL_DIR not set, then check if env var XDG_DATA_HOME is set. If it is set, install python under $XDG_DATA_HOME/uv/python;
  3. If XDG_DATA_HOME is not set either, install python under $HOME/.local/share/uv/python on Unix and %APPDATA%\uv\data\python on Windows

Could anyone tell me if I'm understanding correctly? It should be $UV_PYTHON_INSTALL_DIR rather than $UV_PYTHON_INSTALL_DIR/uv/python, right?
