Comment by @charliermarsh on 2024-11-03 13:09

If you want to install only one group, you need to use dependency-groups rather than project.optional-dependencies:

[project]
dependencies = [
    "cachetools==5.5.0",
]

[dependency-groups]
linters = [
    "ruff==0.7.2",
]

Then you'd run uv sync --only-group linters.
