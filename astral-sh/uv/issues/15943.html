Make --frozen default instead of the other way round

Closed #15943 opened by @ghpqans on 2025-09-19 07:59

Summary

Forgetting to specify --frozen on uv sync, uv run etc. mostly changes uv.lock file. However, the sense of the uv.lock file is to maintain exactly the same set of dependencies over source-trees of different developers working at the same project.

I think it would be better to make it the default to not implicitly change the uv.lock file until explicitly requested.
