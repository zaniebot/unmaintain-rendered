Question

  "torch~=2.7.1",

  "torchaudio~=2.7.1",

[tool.uv.sources]

[[tool.uv.index]]

[[tool.uv.index]]

Comment by @charliermarsh on 2025-06-18 14:58

You should add:

[tool.uv.sources]
torch = { index = "torch-gpu" }
torchvision = { index = "torch-gpu" }
torchaudio = { index = "torch-gpu" }
torchmetrics = { index = "torch-gpu" }

Since torch-gpu is marked as explicit = true, it will only be used for dependencies that are assigned to it explicitly.
