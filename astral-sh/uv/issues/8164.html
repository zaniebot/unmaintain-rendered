uv tool install does not respect index-url from pyproject.toml

#8164 opened by @benjs on 2024-10-14 06:24
Issue opened by @benjs on 2024-10-14 06:24

UV tool does not look into the pyproject.toml for already specified indices.

pyproject.toml:

[tool.uv]
index-url = "https://<my-index-url.com>/pypi/pypi/simple"

command:

uv tool install ruff

uv --version:

uv 0.4.18

Specifying the index url with --index-url worked, but is annoying :) Thanks for this great tool!
