Module 'os' has no attribute 'memfd_create'

Closed #15081 opened by @p-gentili on 2025-08-05 13:24
Issue opened by @p-gentili on 2025-08-05 13:24

Summary

os.memfd_create is a method only supported in "Linux >= 3.17 with glibc >= 2.27" but it's missing from uv python builds (at least 3.10, 3.12).

Step to reproduce

uv python install 3.12
uv run python -c "import os; os.memfd_create"
# AttributeError: module 'os' has no attribute 'memfd_create'

Prove the system supports it

uv run python -c "import ctypes; print(hasattr(ctypes.CDLL('libc.so.6'), 'memfd_create'))"
# True

Platform

Linux 6.14.0-27-generic x86_64 GNU/Linux

Version

uv 0.8.4

Python version

Python 3.12.11
