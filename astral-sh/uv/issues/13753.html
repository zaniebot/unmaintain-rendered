Comment by @charliermarsh on 2025-06-02 00:20

It's not possible to declare swig as an extra build dependency for gymnasium[box2d], but we'll probably support that in the future. (When you added swig on line 29, that would include swig when building your own package (rl), but builds are performed in isolation, so it wouldn't have any affect on the gymnasium[box2d] build.)
