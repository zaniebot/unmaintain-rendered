Closed #12828 opened by @tonnico on 2025-04-11 08:13

Summary

It would be nice to install a local package with different scripts in it.

Example

my project looks like this:

[project]
name = "toolbox"
version = "0.1.0"
description = "Add your description here"
readme = "README.md"
requires-python = ">=3.13"

[project.scripts]
toolbox = "toolbox:app"
tb = "toolbox:app"

[build-system]
requires = ["hatchling"]
build-backend = "hatchling.build"

while

uv tool install .

is working

uv tool install --from=. tb

is not.
