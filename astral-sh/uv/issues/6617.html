virtual workspace dev-dependency did not install extras

Closed #6617 opened by @tonnico on 2024-08-25 20:15
Issue opened by @tonnico on 2024-08-25 20:15

Consider the following pyproject.toml in the root of a project.

[tool.uv]
dev-dependencies = [
    "nox[uv]>=2024.4.15",
]

[tool.uv.workspace]
members = ["foo"]

Until the root is not virtual the uv dependency is installed. After making the root virtual (removing project/build-system section), the extra dependency is removed.

uv init
uv add 'nox[uv]' --dev
cat << EOF > pyproject.toml
[tool.uv]
dev-dependencies = [
    "nox[uv]>=2024.4.15",
]

[tool.uv.workspace]
members = ["foo"]
EOF
