Feature Request

Ability to set prerelease behaviour per index. Unsure if this should be on the command line, in a config or via pyproject.toml (possibly all 3) but any of those would be a great start.

Original question / Use case

I can't currently see any method of doing this but I am not sure if I am just missing something or it's undocumented.
