Summary

The same is true for 3.14 But for 3.15, it only installs python3.15.exe, not python.exe and python3.exe. I couldn't see any warning. I tried adding --verbose, and there was no hint that it even attempted to link python3.exe or python.exe. I couldn't see any information in the docs about some versions being unsuitable for default use.
