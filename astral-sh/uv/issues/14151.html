yes, snap curl is broken (https://github.com/boukendesho/curl-snap/issues/1). We fixed this in github.com/astral-sh/cargo-dist/pull/28, but the fix hasn't shipped yet. You can solve this with:

sudo snap remove curl
sudo apt install curl
