Comment by @charliermarsh on 2025-01-10 02:58

We don't "officially" support it (see here), but I would expect it to work. I obviously can't promise anything, but I would be somewhat surprised if 3.7 broke any time soon. There just aren't very many meaningful changes (from uv's perspective) between 3.7, 3.8, 3.9, etc.
