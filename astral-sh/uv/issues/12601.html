Comment by @zanieb on 2025-04-01 17:04

Your build dependencies are for your project, not yolox. We're working on the ability to extend build dependencies of arbitrary packages, but it's not yet supported.

Since you've already disabled build isolation, you just need to install the dependencies first. e.g., you'd add them to a dependency group, then sync that first

uv sync --only-group yolox-build-deps
uv sync
