Comment by @charliermarsh on 2025-03-08 14:12

A few small things:

  • --app is the default, so you can omit it or include it.
  • If you run uv init project-name, it will create a project in a directory named project-name; if you omit project-name, it will initialize the project in the current directory.
  • You shouldn't need --python $(which python3) unless there's something in your PATH getting in the way. You might prefer --python 3.10 or similar to make sure you get the right version without worrying about where it is.
