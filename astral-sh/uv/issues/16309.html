uv self update can't update beyond version 0.8.19

Open #16309 opened by @UnaiSan on 2025-10-15 06:06
Issue opened by @UnaiSan on 2025-10-15 06:06

Summary

uv self update --verbose gives the following message:

DEBUG uv 0.8.19 (fc7c2f8b5 2025-09-19)
info: Checking for updates...
error: The installation failed. Output from the installer: Downloading uv 0.9.2 (x86_64-pc-windows-msvc)
Exception calling "DownloadFile" with "2" argument(s): "An exception occurred during a WebClient request."
