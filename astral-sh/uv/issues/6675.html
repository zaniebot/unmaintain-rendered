Comment by @zanieb on 2024-08-27 12:03

uv won't manage a system-wide environment yet, we require separate environments. You can use it to install Python, e.g. uv python install. And you can use it to install global tools, e.g., uv tool install hatch to install it on your system instead.

Add the --verbose / -v flag to get more logs, e.g.

uv pip install hatch --system -v
