Hi! It looks like you're on an older version of uv which uses system certificates by default. If you upgrade, we switched to using bundled certificates (like pip) by default in 0.1.18 (https://github.com/astral-sh/uv/pull/2362).

Comment by @zanieb on 2024-05-04 06:08

Hm sorry we don't have a pre-built version of uv for FreeBSD, I'm not sure how hard that would be for us to add. We can track that in https://github.com/astral-sh/uv/issues/3370.

You'll need our development prerequisites to build from source. There are some more details on that in our contributing guide.
