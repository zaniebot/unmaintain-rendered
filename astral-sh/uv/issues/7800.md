_Comment by @charliermarsh on 2024-09-30 02:50_

For whatever reason, [`solace-pubsubplus`](https://pypi.org/project/solace-pubsubplus/#files) only publishes wheels for Python 3.6, and you're using Python 3.11.9. So it's correct to fail here: the project has no source distribution (so you can't build it), nor does it have built distributions for the Python version you're on.
