Comment by @charliermarsh on 2024-10-24 13:52

Yeah you need to pass --system after the command (like uv pip install --system sageattention -- it just has to come after uv pip install). --system is an argument to uv pip install (the subcommand), not uv (the top-level command).

uv [OPTIONS] <COMMAND> is shown because there are some arguments that are global to uv. For example, uv --verbose pip install sageattention is fine, because --verbose is a global argument.
