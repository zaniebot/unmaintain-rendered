Provide a Python native API with bindings to Rust functionality

#11778 opened by @pavelschon on 2025-02-25 14:13
@pavelschon
Summary

I would be great if uv is available also as native python module*), so that all operations with uv could be scripted in python as well, e.g. in DevOps scenarios.

*) Under native module I mean real compiled python/rust module, rather than calling uv binary with subprocess.Popen.

Example 
import uv

uv.python_install(["3.11", "3.12", "3.13"])

uv.venv_create("foo")
