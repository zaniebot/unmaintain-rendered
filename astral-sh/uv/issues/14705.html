Comment by @VBA-N7 on 2025-09-24 09:13

I'm using JFrogGenericArtifacts@1 task. This task is so big for nothing.

- task: JFrogGenericArtifacts@1
  inputs:
    command: 'Upload'
    connection: 'Artifactory connection service'
    specSource: 'taskConfiguration'
    fileSpec: |
      {
        "files": [
          {
            "pattern": "dist/*.whl",
            "target": "$(uploadPath)"
          },
          {
            "pattern": "dist/*.tar.gz",
            "target": "$(uploadPath)"
          }
        ]
      }
    failNoOp: true
  displayName: 'Upload package to Artifactory'
