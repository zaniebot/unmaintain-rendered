Comment by @eegli on 2025-05-27 11:20

The first time you install fastapi via uv pip, you don't install the standard extra. The second time, you run pip install "fastapi[standard]". This pip refers to the user/system installation, uv does not include it in the venv.

Try running uv pip install "fastapi[standard]"in the first place.
