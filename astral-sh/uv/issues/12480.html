Summary

Duplicate packages from uv add and uv pip commands. Minimum steps to reproduce:

uv cache clean

# first download
uv venv test
uv pip install numpy --directory test

# second download, did not use cache
uv init --bare
uv add numpy

Linux 4.18.0

Version

uv 0.6.10

Python version

Python 3.13.2
