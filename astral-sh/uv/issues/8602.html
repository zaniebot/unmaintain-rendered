Comment by @zanieb on 2024-10-28 00:48

I believe uv sync will remove the installation you did with uv pip install because your project is not setup to be "packaged" by uv. You should define a build system or set tool.uv.package = true per https://docs.astral.sh/uv/concepts/projects/#build-systems

Generally, I would not recommend mixing uv sync and uv pip install.
