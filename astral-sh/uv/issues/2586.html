I also faced this error when downloading a large package (1.2GB) from my slow non-PyPI index server.

In my case, UV_CONCURRENT_DOWNLOADS=1 solved the issue. I think we can confidently say the slow downloading causes this error. In my case, I guess my index server is not fast enough to handle many concurrent connections.
