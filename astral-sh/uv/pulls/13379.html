Retain dot-separated wheel tags during cache prune

#13379 opened by @charliermarsh on 2025-05-10 18:28
@charliermarsh

Summary

If a set of wheel tags includes a dot, this code is treating the part after the dot as an extension, and thereby failing to detect that the entry is a symlink to an archive (and thereby removing the archive).

This is all an optimization, so this code just makes it a little targeted: we skip specific known extensions, rather than anything with any extension.

Closes https://github.com/astral-sh/uv/issues/13270.
