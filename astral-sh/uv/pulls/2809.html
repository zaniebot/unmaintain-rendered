Upgrade rs-async-zip to support data descriptors

#2809 opened by @charliermarsh on 2024-04-03 15:06
@charliermarsh

Summary

Upgrading rs-async-zip enables us to support data descriptors in streaming. This both greatly improves performance for indexes that use data descriptors and ensures that we support them in a few other places (e.g., zipped source distributions created in Finder).

Closes #2808.

Comment by @charliermarsh on 2024-04-03 22:09

I've fixed all the perf regressions (and now see improvements, and significant improvements for files that use data descriptors), but I'm now seeing a very rare and sporadic "end of file" failure in some cases for files that contain data descriptors (https://github.com/Majored/rs-async-zip/issues/122#issuecomment-2035686451).
