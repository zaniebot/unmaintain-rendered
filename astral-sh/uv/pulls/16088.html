Make cache control lookups robust to username

#16088 opened by @charliermarsh on 2025-10-01 16:27
@charliermarsh
Summary

We serialize the index to the lockfile without the username, so if we compare based on == and the user includes the username in their pyproject.toml, the check will always fail.

Closes astral-sh/uv#16076.
