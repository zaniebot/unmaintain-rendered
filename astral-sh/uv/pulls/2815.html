Allow package lookups across multiple indexes via explicit opt-in

#2815 opened by @charliermarsh on 2024-04-03 19:43
@charliermarsh

Summary

This partially revives https://github.com/astral-sh/uv/pull/2135 (with some modifications) to enable users to opt-in to looking for packages across multiple indexes.

The behavior is such that, in version selection, we take any compatible version from a "higher-priority" index over the compatible versions of a "lower-priority" index, even if that means we might accept an "older" version.

Closes https://github.com/astral-sh/uv/issues/2775.
