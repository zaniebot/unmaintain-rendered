Respect existing .egg-info files in site packages

#4082 opened by @charliermarsh on 2024-06-06 00:54
Pull request opened by @charliermarsh on 2024-06-06 00:54

Summary

As with other .egg-info and .egg-link distributions, it's easy to support existing .egg-info files. Like pip, we refuse to uninstall these, since there's no way to know which files are part of the distribution.

Closes https://github.com/astral-sh/uv/issues/4059.
