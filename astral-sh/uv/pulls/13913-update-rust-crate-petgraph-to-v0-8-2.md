### [`v0.8.2`](https://redirect.github.com/petgraph/petgraph/blob/HEAD/CHANGELOG.md#082---2025-06-06)

[Compare Source](https://redirect.github.com/petgraph/petgraph/compare/petgraph@v0.8.1...petgraph@v0.8.2)

This minor release fixes several bugs, adds two new algorithms, slightly improves the performance of `maximum_matching`,
adds a tool for parsing graphs from Dot/Graphviz files, and improves the documentation, making it more complete and uniform, as well as clarifying several points.

-   Run Steiner Tree Quickcheck on the connected components to properly support disconnected graphs ([#&#8203;801](https://redirect.github.com/petgraph/petgraph/pull/801))
-   Quickcheck random01 function only outputs 0 ([#&#8203;798](https://redirect.github.com/petgraph/petgraph/pull/798))
