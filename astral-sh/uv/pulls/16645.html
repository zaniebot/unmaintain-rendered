Summary

Similarly to #16529 that adds UV_NO_GROUP, this adds UV_NO_DEFAULT_GROUPS that does the same as --no-default-groups. This can be useful on the CI, to disable default groups on a job without having to set the argument in all commands that could trigger a sync (for instance here).
