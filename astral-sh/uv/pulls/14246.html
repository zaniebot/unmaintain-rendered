Update CONTRIBUTING.md with instructions to format markdown files

#14246 opened by @iamdbychkov on 2025-06-24 16:29
@iamdbychkov

Summary

Current documentation requires contributors to have Node.js/npm installed locally to format Markdown files. This might be problematic for users who don't work with JavaScript ecosystem or users who want to avoid toolchain setup.

This change adds docker-based alternative:

docker run --rm -v .:/src/ -w /src/ node:alpine npx prettier --prose-wrap always --write "**/*.md"

Which mounts current working directory into /src/ inside of a container and also sets working directory (-w) to /src/ so prettier loads .editorconfig.

Test Plan
