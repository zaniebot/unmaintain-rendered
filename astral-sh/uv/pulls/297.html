Respect pip-like Git branch, tag, and commit references

#297 opened by @charliermarsh on 2023-11-02 18:45
Pull request opened by @charliermarsh on 2023-11-02 18:45

We need to parse revisions out from URLs like MyProject @ git+https://git.example.com/MyProject.git@v1.0, per VCS Support. Cargo has the advantage that it uses a TOML table in its configuration, so the user has to specify whether they're fetching a commit, a tag, a branch, etc. We have to instead assume that anything that isn't clearly a commit is either a branch or a tag.

Closes https://github.com/astral-sh/puffin/issues/296.
