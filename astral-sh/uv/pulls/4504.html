Add support for --reinstall and --reinstall-package in uv tool install

#4504 opened by @zanieb on 2024-06-25 01:17
@zanieb

Adds support for --reinstall and --reinstall-package to uv tool install. These are already available via the installer settings, we just respect them now.

--reinstall implies a recreation of the environment and reinstallation of the entry points. --reinstall-package will only update a subset of the environment. If the target package is the one with the entry points, we'll reinstall the entry points. Otherwise, the entry points are not changed.
