Add @generated marker to uv.lock files

#16277 opened by @jackrosenthal on 2025-10-13 03:09
@jackrosenthal
Summary

This adds a comment containing the @generated marker to the top of all uv.lock files. The @generated marker is a widely-adopted convention (see https://generated.at/) that helps tools (like IDEs, code formatters, and code review tools) identify files as generated and handle them appropriately.

Other common tools use this marker in lock files as well (such as Cargo and Poetry).

Fixes #13878.
