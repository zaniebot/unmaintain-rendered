Restructure Git cache to include package name

#588 opened by @charliermarsh on 2023-12-08 00:26
Pull request opened by @charliermarsh on 2023-12-08 00:26

Summary

This PR modifies the Git wheel cache to: (1) use a shorter version of the SHA, to save space; and (2) include the package name, for consistency with all other buckets.

I considered removing the URL hash entirely, and just using the SHA, which would be even more consistent with other buckets. But if we remove the URL, then we won't have separate directories for subdirectories (which are part of the URL).
