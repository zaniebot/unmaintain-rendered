Avoid comparing to system site packages in --dry-run mode

Summary

Right now, uv sync --dry-run returns the interpreter that would've been used to create the environment; so we end up using that interpreter's site-packages, and return changes relative to that interpreter. Instead, we now create a temporary virtual environment and compare against that.

Closes astral-sh/uv#11422.
