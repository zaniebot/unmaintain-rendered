Allow non-file:// paths to serve as --index-url values

#4524 opened by @charliermarsh on 2024-06-25 17:23
Pull request opened by @charliermarsh on 2024-06-25 17:23

Summary

pip allows these with the following logic:

if os.path.exists(location):  # Is a local path.
    url = path_to_url(location)
    path = location
elif location.startswith("file:"):  # A file: URL.
    url = location
    path = url_to_path(location)
elif is_url(location):
    url = location

Closes https://github.com/astral-sh/uv/issues/4510.
