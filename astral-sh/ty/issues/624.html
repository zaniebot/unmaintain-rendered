Reachability is not correct in if-elif-else chains

#624 opened by @sharkdp on 2025-06-10 12:14
@sharkdp

For example:

if True:
    pass
else:
    unknown  # no error here, as expected


if False:
    pass
elif True:
    pass
else:
    # Should be unreachable, but we emit:
    unknown  # error: unresolved-reference
