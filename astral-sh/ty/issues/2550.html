Support enum.global_enum

#2550 opened by @dscorbett on 2026-01-17 20:15
@dscorbett
Summary

enum.global_enum adds an enum’s members to the global namespace. ty reports an error for any use of global constants defined that way. Example:

$ cat >module.py <<'# EOF'
from enum import Enum, auto, global_enum


@global_enum
class E(Enum):
    X = auto()
# EOF

$ cat >main.py <<'# EOF'
from module import X
# EOF

$ ty check main.py --output-format concise
main.py:1:20: error[unresolved-import] Module `module` has no member `X`
Found 1 diagnostic
Version

ty 0.0.12 (4b74e4ded 2026-01-14)
