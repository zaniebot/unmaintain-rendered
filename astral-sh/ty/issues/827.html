Feature request: clear the terminal when --watch is used.

#827 opened by @borissmidt on 2025-07-15 13:44
Issue opened by @borissmidt on 2025-07-15 13:44

Because ty --watch keeps on appending to the current terminal it can be a bit confusing to read its output. Its also hard to find the start of the output of the latest run.

With watchexec, which i assume ty is using behind the scenes, you can set --clear which clears the terminal. Then all you see in the terminal is the latest data of a run making it easier to see which issues are still present.
