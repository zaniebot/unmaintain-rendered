Ty does not remove None from return type after assertion

#1115 opened by @tamir-bahar on 2025-09-02 12:18
Issue opened by @tamir-bahar on 2025-09-02 12:18

Summary

When analyzing the following code:

from typing import reveal_type

def non_optional[T](value: T|None)->T:
    assert value is not None
    return value

def f(x: int|None):
    reveal_type(x)
    y = non_optional(x)
    reveal_type(y)

I expect the revealed type of y to be int, but I get int | None. This behaviour is different from both mypy and pyrefly.
