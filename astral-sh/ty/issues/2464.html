Summary

Given the following code

def my_function() -> str:
    return "Hello World"


__all__ = ["my_function"]

I want to rename my_function to a_function via tys language server.

Expected Result

def a_function() -> str:
    return "Hello World"


__all__ = ["a_function"]

Actual Result

def a_function() -> str:
    return "Hello World"


__all__ = ["my_function"]  # <-- Still "my_" instead of "a_"
