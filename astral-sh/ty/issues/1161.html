NamedTuple-specific methods not part of auto-completions

#1161 opened by @sharkdp on 2025-09-10 10:01
@sharkdp

Start with:

from typing import NamedTuple


class Person(NamedTuple):
    name: str
    age: int | None = None

person = Person("Alice", 25)

and then try to auto-complete on person.<CURSOR>. This should include methods from NamedTupleFallback, e.g. _asdict, but does not.

I think this should be easy to fix by doing something similar to what we do for TypedDicts here, only with NamedTuple and NamedTupleFallback.
