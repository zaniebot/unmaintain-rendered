Respect specified variable type for value of type Any or Unknown

#2148 opened by @kuyugama on 2025-12-21 18:20
@kuyugama

If the value has an Any or Unknown type and variable with that value has specified type - respect the variable type

Current behavior - variable's value type stays it's original type (Any or Unknown) Expected behavior - variable's value type changes to variable specific type

Code I've tested it with:

from typing import Literal, reveal_type, Any


def app() -> Any:
    return "123"


x: Literal["123"] = app()

reveal_type(x)

ty check run output:

   |             ^ `Any`
