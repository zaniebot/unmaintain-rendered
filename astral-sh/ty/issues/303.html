workaround

  1. install https://marketplace.visualstudio.com/items?itemName=maximsmol.vscode-lsp-generic
  2. configure the ty language server in .vscode/settings.json:
    {
  "lsp_generic_client.servers": {
    "ty": {
      "name": "ty",
      "path": "uv",
      "args": ["run", "ty", "server"],
      "documentSelector": ["python"]
    }
  }
}
