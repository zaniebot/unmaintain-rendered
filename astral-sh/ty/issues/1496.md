_Comment by @MichaReiser on 2025-11-07 15:00_

There's no option to suppress an inlay hint (inferred type) within the editor for a specific line. I'm also not aware of any other server that supports this and I'm not convinced that pragma comments are a good use case for this because only some users in a project may have inlay hints enabled.

But we can think about ways to reduce the verbosity. E.g. Pylance only shows inlays for the last unpacked value:


```py
model_instance, model_settings: ModelSettings = create_model_and_settings()
```

which makes the line much shorter
