Display variance on hover over type variables

#1348 opened by @mtshiba on 2025-10-13 17:09
@mtshiba

The variance of PEP-695 style type variables is inferred based on how they are used. It would be useful to be able to see the inferred variance in the editor by hovering over the type variable.

This feature is implemented in pyright/pylance.

By the way, in the playground, nothing seems to be displayed even if we try to hover over the type variable definition of a class.
