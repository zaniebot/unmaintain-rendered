Emit diagnostics for unsupported comparisons involving non-fixed-length tuples

#1741 opened by @AlexWaygood on 2025-12-03 12:20
@AlexWaygood

Pyright, mypy and pyrefly all emit diagnostics on all four comparisons here, but ty currently does not emit any diagnostics on this snippet:

def f(
    a: tuple[int, ...],
    b: tuple[str, ...],
    c: tuple[str]
):
    a < b
    b < a
    a < c
    c < a
Comment by @AlexWaygood on 2025-12-03 12:29

we already emit diagnostics if both operands are fixed-length tuples, e.g.

def f(
    x: tuple[int, int],
    y: tuple[str]
):
    x < y  # error: [unsupported-operator]
