Summary

Example code:

import random

def test(r: tuple[int, int]):
    x = random.randint(*r)
    print(x)

4 |     x = random.randint(*r)
  |         ^^^^^^^^^^^^^^^^^^
5 |     print(x)
  |
info: rule `missing-argument` is enabled by default

Interestingly, this works in the playground, but I'm not sure exactly which version is running there..
