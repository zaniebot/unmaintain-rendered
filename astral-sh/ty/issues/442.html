Hi.

I'm not familiar with Helix myself, but this is a heads-up that diagnostics aren't expected to work because Helix only supports push diagnostics, while ty currently only supports pull diagnostics.

You can still setup ty to get hover and go-to-type-definition but diagnostics will, unfortunately not work at the moment.
