Comment by @sharkdp on 2025-12-19 12:18

The problem here is that client is a property on ServerAccount, but you want to use the imported client in the type hint for _client. See astral-sh/ty#1747 for a related discussion. The short version is that you'll probably want to rename that import (e.g. from acme import client as acme_client).
