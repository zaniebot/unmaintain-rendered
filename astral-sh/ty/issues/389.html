Summary

Hello, I'm having false positive with error[unresolved-reference] in a very specific case of nested functions.

Minimal example:

def bug_inner() -> str:
    if True: # no error without this if
        inner = "x"

        def _inner_fct() -> str:
            if "x" == inner:
                return "x"

        return _inner_fct()

The error:

error[unresolved-reference]: Name `inner` used when not defined
 --> scratch_1425.py:6:23
  |
5 |         def _inner_fct() -> str:
6 |             if "x" == inner:
  |                       ^^^^^
7 |                 return "x"
  |
