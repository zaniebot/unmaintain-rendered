Summary

I think https://github.com/DevinR528/cargo-sort is a cool and useful tool.

Just trying this out to see what people think.

-g keeps groups when sorting tables, which means we can keep ruff and ty dependencies at the top of dependency tables.

Not a huge fan of having to add a root file tomlfmt.toml. This is because the default table order is

table_order = [
    "package",
    "workspace",
    "lib",
    "bin",
    "features",
    "dependencies",
    "build-dependencies",
    "dev-dependencies",
]

And i have moved "features" to the bottom in the configuration.

table_order = [
    "package",
    "workspace",
    "lib",
    "bin",
    "dependencies",
    "build-dependencies",
    "dev-dependencies",
    "features",
]
