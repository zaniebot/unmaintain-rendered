Summary

This pull request implements the R1716 pylint rule: pylint documentation.

It checks boolean and operations that can can be chained

a = int(input())
b = int(input())
c = int(input())
if a < b and b < c:
    pass

And suggest to combine them:

a = int(input())
b = int(input())
c = int(input())
if a < b < c:
    pass

Test Plan

I have added test cases, and checked the ruff ecosystem results.
