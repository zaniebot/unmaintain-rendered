Comment by @harupy on 2022-12-24 04:08

I wrote this shell script and have been using it :)

for f in $(find . -name '*.snap.new'); do
    new_f=$(echo $f | sed 's/\.snap\.new$/.snap/')
    echo $new_f
    cat $f | grep -v '^assertion_line' > $new_f
    rm $f
done
