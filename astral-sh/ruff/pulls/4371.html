List rule changes in ecosystem

#4371 opened by @calumy on 2023-05-11 11:14
@calumy

Currently, the ecosystem check lists all changes between two versions of ruff as a diff. This PR counts the changes for each rule so the impact of the change can be determined.

A use case where this could be useful is when adding a set of new rules to determine if one rule will cause a particularly large number of new rule violations to be raised. For example, the addition of the new flake8-todo rules in #3921 raised a discussion about the impact of the rule TDO003 which checks that TODOs have links present.
