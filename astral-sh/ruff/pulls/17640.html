[red-knot] Use 101 exit code when there's at least one diagnostic with severity 'fatal'

#17640 opened by @MichaReiser on 2025-04-26 09:56
@MichaReiser
Summary

We now capture panics during checking and emit a diagnostic with severity Fatal but the CLI still exits with ExitStatus::Failure (1) which makes it indistinguishable from a normal: there are diagnostics.

This PR changes the implementation to return ExitStatus::Eror (which we code to 2)
