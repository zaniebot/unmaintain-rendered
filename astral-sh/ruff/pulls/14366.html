Update insta snapshots

#14366 opened by @MichaReiser on 2024-11-15 18:02
@MichaReiser
Summary

Cargo insta made updates to its snapshot formats. The new format isn't applied unless there are other changes to the snapshot. I've found this annoying when a snapshot test failed, I accepted the changes, and then fixed the regression because I then ended up with an "empty" snapshot change that updated the snapshot to the new layout.

This PR updates all snapshots to use the new insta snapshot layout
