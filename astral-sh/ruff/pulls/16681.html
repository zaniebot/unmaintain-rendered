Pull request opened by @MichaReiser on 2025-03-12 15:42

Summary

This PR deprecates UP038. Using PEP 604 syntax in isinstance and issubclass calls isn't a recommended pattern (or community agreed best practice) and it negatively impacts performance.

Resolves https://github.com/astral-sh/ruff/issues/7871
