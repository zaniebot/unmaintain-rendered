[ty] Add support for PEP 800

#20084 opened by @AlexWaygood on 2025-08-25 17:36

Summary

This PR adds support for PEP 800. Since ty's implementation of disjointness was the original inspiration for the PEP, nearly everything was already implemented; this PR mainly consists of renaming internal concepts to match up with the terms used by the PEP, and hooking things up so that we recognize the effect the new @disjoint_base decorator has.
