Add support for providing command-line arguments via argfile

#4087 opened by @charliermarsh on 2023-04-25 05:41
Pull request opened by @charliermarsh on 2023-04-25 05:41

Summary

This PR uses the argfile crate to allow users to provide additional command-line arguments as a separate argument file.

For example, one might create file like list.txt:

foo.py
bar.py

Then, invoking ruff @list.txt would be equivalent to ruff foo.py bar.py.

Closes #4079.
