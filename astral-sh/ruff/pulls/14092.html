Avoid cloning Name when looking up function and class types

#14092 opened by @MichaReiser on 2024-11-04 14:44
@MichaReiser

Summary

Uses Salsa's new Lookup functionality for FunctionType and ClassType names. The Lookup trait allows looking-up interned values by a borrowed value. In this case we can lookup the interned type using the borrowed str and only allocate the owned `Name`` when the type doesn't already exist.

This was part of https://github.com/astral-sh/ruff/pull/14087 but I extracted it into its own PR to see if it is the reason for the significant incremental perf improvement.
