Summary

one can't access the module docstring for a notebook which makes a notebook not having a module level docstring. @dhruv

This PR handles strings at the top of a cell as non-docstrings. Handling them as docstrings has the undesired result that one might have multiple "docstrings" in a single notebook, which is probably not what you want.

We don't need to gate this behind preview because it doesn't change any formatted code.
