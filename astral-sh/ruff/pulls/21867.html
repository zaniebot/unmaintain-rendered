Summary

Adds support for checking files without extensions if they're passed by name on the CLI or if the user selects Python as the language in their editor.

Fixes https://github.com/astral-sh/ty/issues/1692

Note: We don't yet support excluding a file that has a .py extension if the user selects a non-Python language in their editor but I consider this a separate change.
