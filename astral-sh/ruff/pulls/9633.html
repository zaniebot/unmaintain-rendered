help: enable auto-wrapping of help output

#9633 opened by @BurntSushi on 2024-01-24 14:34
Pull request opened by @BurntSushi on 2024-01-24 14:34

Previously, without the 'wrap_help' feature enabled, Clap would not do any auto-wrapping of help text. For help text with long lines, this tends to lead to non-ideal formatting. It can be especially difficult to read when the width of the terminal is smaller.

This commit enables 'wrap_help', which will automatically cause Clap to query the terminal size and wrap according to that. Or, if the terminal size cannot be determined, it will default to a maximum line width of 100.

Ref https://github.com/astral-sh/ruff/pull/9599#discussion_r1464992692
