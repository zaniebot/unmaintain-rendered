Summary

This PR implements formatting of ExprStarred nodes (call(*args)).

We deviate from black when formatting a trailing * comment. I opted for the "easiest" solution which is to simply make the comment a leading comment of the whole expression (requires no special handling in the formatter). We can revisit the formatting if this happens to come up frequently (I hope no-one puts a comment there).
