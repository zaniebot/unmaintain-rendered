## Summary

This moves the `shared_traits.rs` file to `ruff_formatter` and includes it in `ruff_python_formatter` with `include!` because it's supposed to be shared by all language specific implementations.

I used this chance to introduce a `prelude` file in `ruff_python_formatter` to reduce the repetitive imports in every file.
