143 | [tool.ruff.preview]
    | ^^^^^^^^^^^^^^^^^^^
invalid type: map, expected a boolean

Test Plan

Tested running ruff format with the following in my pyproject.toml:

[tool.ruff.format]
preview = true

and it worked properly (using preview rules for formatting).
