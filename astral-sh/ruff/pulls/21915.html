Summary

This PR makes our ty_ide snapshot tests for find references, go to definition, go to declaration, and go to type definition more succingt by:

  • Don't repeat the source for every found go to definition target. They're all the same!
  • Use a single diagnostic for all results with secondary annotations, allowing multiple results to be rendered into the same code frame.
