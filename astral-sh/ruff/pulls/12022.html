Summary

This PR fixes the red-knot tests when run in release mode.

The problem was that the semantic indexer code contained code with side effects inside of debug_assert_eq calls, that get stripped (entirely) from release builds, removing the side effects.

This PR moves the side-effects out from the debug_assert_eq calls.

Fixes https://github.com/astral-sh/ruff/issues/12018
