- [#&#8203;440](https://redirect.github.com/Peternator7/strum/pull/440): Implement a `suffix` attribute for serialization of enum variants.

  ```rust

  #[strum(suffix=".json")]
  #[strum(serialize_all="snake_case")]
  enum StorageConfiguration {
    PostgresProvider,
    S3StorageProvider,
    AzureStorageProvider,
  }

  fn main() {
    let response = SurveyResponse::Other("It was good".into());
  ```
