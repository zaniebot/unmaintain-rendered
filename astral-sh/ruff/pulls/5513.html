Summary

VS Code extensions are recommended to exclude .vscode and site-packages. Black also now omits .vscode, .pytest_cache, and .ipynb_checkpoints by default. Omitting .pyenv is similar to omitting virtual environments, but really only matters in the context of VS Code (see: https://github.com/astral-sh/ruff/discussions/5509).

Closes: #5510.
