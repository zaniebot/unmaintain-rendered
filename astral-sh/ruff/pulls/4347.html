Enforce max-doc-length for multi-line docstrings

#4347 opened by @charliermarsh on 2023-05-10 14:51
@charliermarsh
Summary

Our doc_lines vector is intended to contain the start locations of all doc lines. However, for multi-line docstrings, we're only including the start of the first line, so we're missing overlong lines in multi-line docstrings that occur beyond the first line in the docstring.

This PR modifies the collection code to include the start of all lines in the docstring.

Closes #4344.
