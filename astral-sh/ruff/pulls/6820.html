Respect own-line leading comments before parenthesized nodes

#6820 opened by @charliermarsh on 2023-08-23 17:09
@charliermarsh
Summary

This PR ensures that if an expression has an own-line leading comment before its open parentheses, we render it as such.

For example, given:

[ # foo
    # bar
    ( # baz
        1
    )
]

On main, we format as:

[  # foo
    (
        # bar
        # baz
        1
    )
]

As of this PR, we format as:

[  # foo
    # bar
    (  # baz
        1
    )
]
