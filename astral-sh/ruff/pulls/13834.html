Simplify iteration idioms

#13834 opened by @AlexWaygood on 2024-10-20 21:07
@AlexWaygood
Summary

This PR removes unnecessary uses of .as_ref(), .iter(), &** and similar, mostly in situations when iterating over variables. Many of these changes are only possible following #13826, when we bumped our MSRV to 1.80: several useful implementations on &Box<[T]> were only stabilised in Rust 1.80. Some of these changes we could have done earlier, however.
