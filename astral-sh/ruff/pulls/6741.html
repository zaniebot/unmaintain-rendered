Summary

There's a case where a trailing comment ins't formatted like in Black. Consider the following code:

match x:
    case (
            None # trailing
            ):
        ...

Black formats to

match x:
    case (None):  # trailing
        ...

And Ruff will format to

match x:
    case (
        None  # trailing
    ):
        ...

Closes #6557
