[ty] Remove 'pre-release software' warning

#20817 opened by @MichaReiser on 2025-10-12 05:44
@MichaReiser

Summary

This PR removes the "ty is pre-release software..." warning from the CLI. I think it's dramatically underselling where ty is today (compared to when we released the alpha earlier this year). While there are still a few crashes and some missing language features, I think that's covered sufficiently by the fact that ty is released as an alpha version.
