## Summary

The existing `Visitor` trait traverses the tree in evaluation-order. In the formatter, 
we iterate over the comments and try to attach them to the right node. This only works 
if we iterate the nodes in pre-order. 

This PR introduces a new `PreorderVisitor` trait that allows pre-order traversal of the AST nodes.
