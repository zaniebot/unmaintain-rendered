Mark wildcard imports as unused when all references are resolved

#9032 opened by @charliermarsh on 2023-12-07 01:58
Pull request opened by @charliermarsh on 2023-12-07 01:58

Summary

If a file contains, e.g., from foo import *, but all references are resolved, we should mark the import as unused (similar to Flake8).

This change is gated behind preview for now.

Closes https://github.com/astral-sh/ruff/issues/9028.

Comment by @charliermarsh on 2023-12-07 03:10

It would not remove the import. Given:

from "a" import b, c

print(b)

from foo import *

print(a) # we can resolve `a` but we don't know if `foo` rebound `a`

When we get to print(a), we would be unable to resolve the symbol to anything defined in the file, and so we would assume that a comes from from foo import *, and not remove it.
