[ty] Update salsa

#22072 opened by @MichaReiser on 2025-12-19 09:28
@MichaReiser

Summary

Salsa now enforces that interned structs and many-argument queries implement salsa::Update (or have a &'static lifetime). This prevents unsoundness bugs, but requires us to derive salsa::Update in a few more places.

Note, this requires https://github.com/salsa-rs/salsa/pull/1041 or we have to manually implement salsa::Update for the tuple types, which is a bit painful.
