Summary

By default, Monaco sorts the completion items by the label but we want to preserve the order returned by the completions functions. Ideally, we could just disable sorting but I couldn't find a way to do this.

This PR sets the sortText which monaco uses for sorting. It's a bit awkward because it's a string (which gets compared in alphabetic ordering) But the basic idea is that we use the padded index as sort key.
