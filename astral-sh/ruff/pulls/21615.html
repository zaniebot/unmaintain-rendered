[ty] Add hint about resolved Python version when a user attempts to import a member added on a newer version

#21615 opened by @AlexWaygood on 2025-11-24 14:54
@AlexWaygood
Summary

Fixes astral-sh/ty#1620. #20909 added hints if you do something like this and your Python version is set to 3.10 or lower:

import typing

typing.LiteralString

And we also have hints if you try to do something like this and your Python version is set too low:

from stdlib_module import new_submodule

But we don't currently have any subdiagnostic hint if you do something like this and your Python version is set too low:

from typing import LiteralString

This PR adds that hint!
