[ty] Use generator over list comprehension to avoid cast

#21748 opened by @MichaReiser on 2025-12-02 07:33
@MichaReiser
Summary

Suggested by @sharkdp. ty (incorrectly) infers list[Unknown] when using asyncio.gather because the list comprehension is unioned with | Unknown (because it's a list that could be modified but never will).

Using a generator avoids the | Unknown, making ty infer the correct return type.
