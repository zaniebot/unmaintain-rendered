Handle comments on open parentheses in with statements

#6515 opened by @charliermarsh on 2023-08-12 00:26
Pull request opened by @charliermarsh on 2023-08-12 00:26

Summary

This PR adds handling for comments on open parentheses in parenthesized context managers. For example, given:

with (  # comment
    CtxManager1() as example1,
    CtxManager2() as example2,
    CtxManager3() as example3,
):
    ...

We want to preserve that formatting. (Black does the same.) On main, we format as:

with (
    # comment
    CtxManager1() as example1,
    CtxManager2() as example2,
    CtxManager3() as example3,
):
    ...

It's very similar to how StmtImportFrom is handled.

Note that this case isn't covered by the "parenthesized comment" proposal, since this is a common on the statement that would typically be attached to the first WithItem, and the WithItem itself can have parenthesized comments, like:

with (  # comment
    (
        CtxManager1()  # comment
    ) as example1,
    CtxManager2() as example2,
    CtxManager3() as example3,
):
    ...
