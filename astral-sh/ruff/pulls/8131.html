Remove --line-length option from format command

#8131 opened by @MichaReiser on 2023-10-23 09:27
@MichaReiser

Summary

This PR removes the undocumented --line-length setting from the format command. The main reason is that we should either support all formatter options or none of them. I would prefer not supporting formatter options via the CLI.

We can reintroduce the option in the beta based on user feedback.
