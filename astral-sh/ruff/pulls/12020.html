Redirect F509 to PLE1300

Summary

This PR removes the F509 rule and redirects it to PLE1300

For context, PLE1300 looks at both %-style strings and .format strings while F509 only looks at % style strings. Basically, F509 is a subset of PLE1300 which means we should remove F509 instead.

This PR also updates the documentation for PLE1300 with "Use instead" and "References" section.
