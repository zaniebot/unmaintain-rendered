Consider : to terminate parenthesized with items

#11775 opened by @dhruvmanila on 2024-06-06 11:44
@dhruvmanila
Summary

This PR is a follow-up to this discussion (astral-sh/ruff#11770#discussion_r1628917209) which adds the : token in the terminator set for parenthesized with items.

The main motivation is to avoid parsing too much in speculative mode. This is evident with the following before and after parsed with items list for the following code:

with (item1, item2:
    foo
Before (3 items) After (2 items)
