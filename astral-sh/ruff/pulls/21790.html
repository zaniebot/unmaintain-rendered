[ty] Ensure rename CursorTest calls can_rename before renaming

#21790 opened by @MichaReiser on 2025-12-04 12:48
@MichaReiser
Summary

This PR ensures that CursorTest::rename calls can_rename before renaming a symbol.

It happened multiple times that I added a test that works fine in CursorTest but that then doesn't work in VS Code because can_rename returns nope.

This change ensures that our testing infrastructure more closely resembles the flow used by actual LSP clients.
