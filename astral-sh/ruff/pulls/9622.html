Summary

Black's snapshot testing uses a # flags: --option=value comment at the top of snapshot files to configure CLI arguments. Black's testing infrastructure strips that comment before calling black, except when the flags comment contains the --line-ranges flag, in which case the comment is preserved.

This PR updates our import script to preserve the # flags comment if it contains the --line-ranges flag similar to Black's testing infrastructure. It reimports the Black test cases and updates the snapshots. This PR contains no change outside our testing infrastructure.
