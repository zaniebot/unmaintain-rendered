Summary

Log files that take long to type check to ease debugging cases like https://github.com/astral-sh/ty/issues/1337

This won't help with cases where ty hangs completely and it can also be inaccurate in cases where checking one files requires inferring types for many third party libraries but I think it's better than nothing. I picked the 100ms limit arbitrarily.

Test Plan

Tested that the logs show for some files that I know that long to type check (in debug).
