Summary

This PR fixes a bug in Red Knot's file watching where a new non-project file (a file created outside the project's root) was incorrectly considered as being part of the project.

Testing whether the new file's path starts with the project's root path is sufficient because neither Ruff nor Red Knot follow symlinks when traversing the project's directory tree.
