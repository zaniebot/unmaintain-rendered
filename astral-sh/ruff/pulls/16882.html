Summary

Extend pep8-naming to catch mixed‑case identifiers introduced via structural pattern matching:

  • N806: variables bound by a match pattern inside functions
  • N815: variables bound by a match pattern inside classes

Previously these names were ignored since they weren’t visited during expression store analysis. Add a pattern walker that binds pattern names and runs the same naming checks based on the current scope. Test fixtures and snapshots for N806, N815 and N816 have been updated with match cases.
