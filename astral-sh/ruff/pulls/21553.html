Summary

Add support for generic PEP 613 type aliases and generic implicit type aliases:

from typing import TypeVar

T = TypeVar("T")
ListOrSet = list[T] | set[T]

def _(xs: ListOrSet[int]):
    reveal_type(xs)  # list[int] | set[int]
