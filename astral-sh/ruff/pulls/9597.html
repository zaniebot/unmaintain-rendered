Summary

Implement rule mutable-fromkeys-value (RUF023).

Autofixes

dict.fromkeys(foo, [])

to

{key: [] for key in foo}

The fix is marked as unsafe as it changes runtime behaviour. It also uses key as the comprehension variable, which may not always be desired.

Closes #4613.
