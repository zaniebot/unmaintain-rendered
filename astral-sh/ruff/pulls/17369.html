Summary

This PR adds a script to generate a pgo optimized binary of ruff, see #7055 for more detail. I initially only implemented the script on x86_64-unknown-linux-gnu so people can try it out, if we think this is more interesting it should probably be added to the release pipeline. from what I can see you get a general 3-10% improvement of runtime speed then running the pgo-binaries, as well as a somewhat smaller binary size.

The script fetches a large popular python codebases and uses that as training data (see pgo_profile.json),

this is an example of the performance for a cold run on pytorch's source, where the pgo is 12% faster.
