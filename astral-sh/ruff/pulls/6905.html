Summary

There was only one case where I couldn't figure it out how to format properly, which is the following

match x:
    case (
        (a)
        | # trailing
        ( b )
    ):
        ...

That code is being formatted to this

match x:
    case (
        (
            a  # trailing
        )
        | (b)
    ):
        ...

But it should be formatted to this

match x:
    case (
        (a)  # trailing
        | (b)
    ):
        ...

Closes #6643
