You could do tagged unions of tuples next if you fancy it? E.g.

```py
from typing import Literal

class A: ...
class B: ...
class C: ...

def f(x: tuple[Literal["tag1"], A] | tuple[Literal["tag2"], B, C]):
    if x[0] == "tag1":
        reveal_type(x[1])  # revealed: A
        reveal_type(len(x))  # revealed: Literal[2]
        x[2]  # error: [index-out-of-bounds]
    else:
        reveal_type(x[1])  # revealed: B
        reveal_type(len(x))  # revealed: Literal[3]
        reveal_type(x[2])  # revealed: C
```

it should work in a very similar way to our tagged-union narrowing for `TypedDict`s.
