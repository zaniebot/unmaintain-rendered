Use a separate repo for pre-commit

#229 opened by @charliermarsh on 2022-09-20 03:06

Resolves #225.

By defining the pre-commit configuration in this repo, users had to build ruff from source, which requires Cargo. By putting the pre-commit configuration in a standalone repo that depends on ruff, we can leverage PyPI's prebuilt wheels.
