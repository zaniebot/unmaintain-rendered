Fix parenthesized detection for tuples

#6599 opened by @charliermarsh on 2023-08-15 16:28
@charliermarsh
Summary

This PR fixes our code for detecting whether a tuple has its own parentheses, which is necessary when attempting to preserve parentheses. As-is, we were getting some cases wrong, like (a := 1), (b := 3)) -- the detection code inferred that this was parenthesized, and so wrapped the entire thing in an unnecessary set of parentheses.
