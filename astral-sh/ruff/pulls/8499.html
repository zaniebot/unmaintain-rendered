Allow collapsed-ellipsis bodies in other statements

#8499 opened by @charliermarsh on 2023-11-06 00:25
@charliermarsh
Summary

Black and Ruff's preview styles now collapse statements like:

from contextlib import nullcontext

ctx = nullcontext()
with ctx: ...

Historically, we made an exception here for classes (astral-sh/ruff#2837). This PR extends it to other statement kinds for consistency with the formatter.

Closes astral-sh/ruff#8496.
