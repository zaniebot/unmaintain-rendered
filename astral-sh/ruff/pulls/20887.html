Remove release CI job

#20887 opened by @MichaReiser on 2025-10-15 10:05
@MichaReiser
Summary

We added the release CI job to catch issues with conditionally compiled code (debug, cfg(debug_assertions) back in April 2024 (astral-sh/ruff#11182).

But, we've since then also added cargo-test-linux-release (astral-sh/ruff#14604), which not only compiles with the release profile but also runs all tests

https://github.com/astral-sh/ruff/blob/3d2ece2f42f24b0c18968d3a624d37eeb37525ff/.github/workflows/ci.yaml#L300-L331

That's why I think we can safely remove the release CI Job, as it's covered by cargo-test-linux-release
