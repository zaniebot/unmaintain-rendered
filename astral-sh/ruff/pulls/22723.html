Summary

Another standalone refactor that has no functional changes, also pulled out of #22718.

This PR changes the DynamicNamedTupleLiteral::fields field to have type Box<[NamedTupleField<'db>]> rather than Box<[(Name, Type<'db>, Option<Type<'db>)]>. This makes Clippy a lot happier, makes the code easier to read IMO, and means that rustfmt significantly reduces the indentation of a large chunk of our code in infer/builder.rs.

The diff is much easier to read if you select the "hide whitespace changes" option on GitHub!
