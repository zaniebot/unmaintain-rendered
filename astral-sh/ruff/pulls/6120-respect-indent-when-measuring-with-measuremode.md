## Summary

This PR contains two smaller Printer improvements:

1. The printer didn't queue an indent after a newline when measuring if all lines fits. This means, it measured the lines after the first line without any indent. 
2. Remove the unused `pending_space` state variable

This PR contains another small bugfix where `ExprCall` and `ExprAttribute` passed the wrong parent in `NeedsParentheses` (their parent instead of self)
