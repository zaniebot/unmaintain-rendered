Allow named expressions in __all__ assignments

#7673 opened by @charliermarsh on 2023-09-27 04:17
Pull request opened by @charliermarsh on 2023-09-27 04:17

Summary

This PR adds support for named expressions when analyzing __all__ assignments, as per https://github.com/astral-sh/ruff/issues/7672. It also loosens the enforcement around assignments like: __all__ = list(some_other_expression). We shouldn't flag these as invalid, even though we can't analyze the members, since we know they evaluate to a list.

Closes https://github.com/astral-sh/ruff/issues/7672.
