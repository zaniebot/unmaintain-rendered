Trying to understand why PLW0120 ("Else clause on loop without a break statement, remove the else and de-indent all the code inside it") is being raised.

Here's a simple example which causes PLW0120, even though it does have a break:

for page in range(1, 10):
    if page < 5:
        print(page)
    else:
        print("error")
        break
else:
    print("OK")

When the break is moved inside the if, PLW0120 goes away.
