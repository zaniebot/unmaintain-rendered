Comment by @tjkuson on 2025-01-10 01:46

A trailing comma should force ruff format to leave arguments on separate lines.

For example,

def fn(
    foo,
    bar,
):
    pass

will remain unchanged, as will

print(
    "hello",
    "world",
)
