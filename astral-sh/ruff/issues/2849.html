Comment by @JonathanPlasse on 2023-02-13 11:46

Is it intentional to have # noqa: E501 inside the string? Ruff does not read noqa inside strings. You must place the noqa after the string for it to work.

    """
     >>> UsernameHandler(20).format_username('Foo[COUNTER2][COUNTER2]')  # noqa: E501 # doctest: +IGNORE_EXCEPTION_DETAIL
    """  # noqa: E501
Comment by @charliermarsh on 2023-02-13 13:39

That’s correct — we don’t respect noqa comments inside of strings.
