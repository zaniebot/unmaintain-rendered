Suppress ANN401 for dynamically typed *args and **kwargs.

Issue opened by @tekumara on 2022-11-11 05:09

Now that ANN401 has landed, would be nice to support allow-star-arg-any which is common idiom, eg:

def mocked_fn(*args: Any, **kwargs: Any) -> None:
   ...
Comment by @andersk on 2022-11-11 05:19

This is supported.

[tool.ruff.flake8-annotations]
allow-star-arg-any = true

(Note though that *args: object, **kwargs: object or ParamSpec is often a safer alternative.)
