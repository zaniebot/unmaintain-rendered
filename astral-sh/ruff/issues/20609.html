`warning: --fix is incompatible with --add-noqa` when using `--diff`

#20609 opened by @smurfix on 2025-09-28 15:38
Issue opened by @smurfix on 2025-09-28 15:38

Summary

It is not possible to see what an add-noqa operation would do before actually doing it.

$ ruff check --select D101,D102 --add-noqa --diff src/
warning: --fix is incompatible with --add-noqa.
$

IMHO: ruff either complains, or it generates fix-ups. In the latter case, whether ruff (a) rewrites the code or (b) adds #noqa should be orthogonal to whether (x) changes are applied or (y) a diff is emitted; ax ay bx by should all be possible.

Version

0.13.2
