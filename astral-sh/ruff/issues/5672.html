Formatter: Don't insert a newline between if and parentheses in list comprehensions

Closed #5672 opened by @konstin on 2023-07-11 07:34
Issue opened by @konstin on 2023-07-11 07:34

We format this black formatted code from django django/apps/config.py

configs = [
    (name, candidate)
    for name, candidate in members
    if (
        issubclass(candidate, cls)
        and candidate is not cls
        and getattr(candidate, "default", True)
    )
]

as

configs = [
    (name, candidate)
    for name, candidate in members
    if
    (
        issubclass(candidate, cls)
        and candidate is not cls
        and getattr(candidate, "default", True)
    )
]

We should change our formatting to also not emit the newline before the parentheses.
