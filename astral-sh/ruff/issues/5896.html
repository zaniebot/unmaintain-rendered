lambda arguments have the wrong range and cause unstable formatting

#5896 opened by @konstin on 2023-07-19 19:26
Issue opened by @konstin on 2023-07-19 19:26

This example causes unstable formatting:

y=lambda:{
#
}

--- Formatted once
+++ Formatted twice
@@ -1,3 +1,3 @@
 y = (
-    lambda: {}#
+    lambda: {}  #
 )
---

Formatted once:
---
y = (
    lambda: {}#
)
---

Formatted twice:
---
y = (
    lambda: {}  #
)
---

The comment gets attached to the lambda arguments instead of the body expression where it should be:

{
    Node {
        kind: Arguments,
        range: 2..14,
        source: `lambda:{⏎`,
    }: {
        "leading": [],
        "dangling": [
            SourceComment {
                text: "#",
                position: OwnLine,
                formatted: true,
            },
        ],
        "trailing": [],
    },
}
