Hi @ayjayt

As @dylwil3 said, it's a philosophical decision that the formatter has only a few options. Many users see this as its main feature because it prevents them from having to make any formatting decisions. I always liked the motivation outlined in Prettier's philosophy documentation.

This doesn't mean that we don't plan to add any new options, it only means that we have a relatively high bar for adding new formatter options and the options are mainly limited to configurations that don't change the code style, but some detail like what quotes you prefer.
