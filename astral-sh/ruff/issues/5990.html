Format VS Code-style # %% cell delimiters

VS Code Python has a very nice feature called Interactive Window that allows to run regular Python scripts in a Jupyter-like interface.

Cells are delimited by # %%. For readability, I like to keep 2 blank lines above every cell delimiter (except if at the top of a file). I wrote format-ipy-cells to auto-format that as well as delete empty cells and enforce comment spacing when on the same line as a cell delimiter.

Are these rules that could be integrated into ruff?
