Comment by @charliermarsh on 2023-01-03 18:24

Ahh, this has to do with the way that select and ignore are resolved.

Can you instead try:

[tool.ruff.pydocstyle]
convention = "numpy"  # Accepts: "google", "numpy", or "pep257"

[tool.ruff]
extend-select = ["D"]
extend-ignore = ["D203"]

Or:

[tool.ruff.pydocstyle]
convention = "numpy"  # Accepts: "google", "numpy", or "pep257"

[tool.ruff]
select = ["D"]
ignore = ["D203"]

(The extend is always applied after the non-extend. So above, you're removing D203, then adding all the D rules on top of that.)
