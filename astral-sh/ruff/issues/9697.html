isort ability to treat comments as code

#9697 opened by @yasirroni on 2024-01-30 05:43
@yasirroni

isort support ability to treat comments as code pycqa/isort#1357. It is very useful in .py files generated from .ipynb that each section of import separated by comment # %%

Thus,

import os

import cvxpy as cp
import matplotlib.pyplot as plt
import numpy as np
import pandas as pd

# %%  >>> This should make all import below this treated as separate isort task
import sys
import traceback
import warnings
