`SIM103` should detect implicit `else`

Closed #10402 opened by @charliermarsh on 2024-03-14 02:35
Issue opened by @charliermarsh on 2024-03-14 02:35

See: https://github.com/sbdchd/flake8-pie#pie801-prefer-simple-return.

In that example:

# error
def main():
    if foo > 5:
        return True
    return False

# error
def main():
    if foo > 5:
        return True
    else:
        return False

# ok
def main():
    return foo > 5

We catch the second case, but not the first.
