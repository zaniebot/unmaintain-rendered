allow BLE001 if traceback is generated?

Closed #2212 opened by @spaceone on 2023-01-26 18:20
Issue opened by @spaceone on 2023-01-26 18:20

BLE001 allows

try:
        foo
except Exception:
        raise

but disallows:

try:
        foo
except Exception:
        print(traceback.format_exc())
        logger.log(traceback.format_exc())
        logger.exception('something is broken')

I think generating a traceback is an indication of handling the exception properly.
