I get a E999 SyntaxError: invalid syntax. Got unexpected token whenever structural pattern matching occurs. e.g.:

def http_error(status):
    match status:
        case 400:
            return "Bad request"
        case 404:
            return "Not found"
        case 418:
            return "I'm a teapot"
        case _:
            return "Something's wrong with the internet"

I'm running Python 3.10.9
