Comment by @MichaReiser on 2024-01-19 08:37

You can run ruff's formatter by using ruff format <paths>. It only formats your code. It won't run any lint rules.

The VS code extension doesn't yet support your use case but we plan to add a setting soonish. You can follow https://github.com/astral-sh/ruff-vscode/issues/329 to get informed once it's done.
