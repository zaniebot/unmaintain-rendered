Multilingual support for rules D400 and D415

Open #16962 opened by @gakugaku on 2025-03-25 04:48
Issue opened by @gakugaku on 2025-03-25 04:48

Summary

Overview

The rules D400 (missing trailing period) and D415 (missing terminal punctuation) in Ruff currently target English only. They do not support terminal punctuation for other languages. For example, a Japanese comment using 。 at the end may be flagged incorrectly.

Proposed Changes

  • Update the logic for D400 and D415 to support multiple languages.
  • Allow configuration of language-specific terminal punctuation (e.g., English: ., Japanese: 。).

Example for D400/D415 in English:

def average(values: list[float]) -> float:
    """Return the mean of the given values."""

Example in Japanese

def average(values: list[float]) -> float:
    """平均値を返します。"""
