Treat each match case as its own scope

#3099 opened by @charliermarsh on 2023-02-21 21:49
@charliermarsh

Given:

def f():
    match Car("Toyota", "Corolla"):
        case Car("Toyota", model):
            print(model)
        case Car(make, "Corolla"):
            print(make)
        case Car(_, _):
            print(make, model)

The (make, model) in the final case should be flagged as undefined.
