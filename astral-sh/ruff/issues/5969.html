[rule suggestion] simplify `if x, if not x` to `if/else`

Closed #5969 opened by @ScDor on 2023-07-22 07:05
Issue opened by @ScDor on 2023-07-22 07:05

Instead of

if x:
    ...
if not x:
    ...

Use

if x:
    ...
else:
    ...

A little silly, but improves code readability.
