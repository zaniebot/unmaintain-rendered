Summary

Pylint's W9006 rule, missing-raises-doc, checks whether or not a developer has documented manually raised Exceptions in the relevant docstring. This practice not only improves the clarity and maintainability of the code by making the possible error conditions explicit in the documentation, but also helps users of the function/method understand how to properly handle or anticipate these exceptions.
