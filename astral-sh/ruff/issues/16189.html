Description

debugger (T100) doesn’t flag sys.breakpointhook and sys.__breakpointhook__ in Ruff 0.9.6. It should, because T100 does flag breakpoint, which just calls sys.breakpointhook, and there is no reason to treat them differently. Also, Pylint’s forgotten-debug-statement (W1515), which in #970 is mapped to T100, flags sys.breakpointhook.
