Formatter incompatibility: ellipses are not inlined in .py files

#7440 opened by @charliermarsh on 2023-09-16 16:40
@charliermarsh

Given:

def f(): ...

def f():
    ...

def f():
    """Docstring"""
    ...

Recent versions of Black format as:

def f(): ...


def f(): ...


def f():
    """Docstring"""
    ...

While Ruff formats as:

def f():
    ...


def f():
    ...


def f():
    """Docstring"""
    ...

This changed in Black 23.9.0 (psf/black#3796).
