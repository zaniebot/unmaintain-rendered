Kudos for this great tool, it's more than useful for so many projects. I noticed that with the latest release, ruff is throwing errors on RUF029 on async interators. The snippet below:

import asyncio
from typing import Tuple


async def example(time: int) -> Tuple[int, int, int]:
    await asyncio.sleep(time)
    return (1, 2, 3)

    vals = [str(val) for val in await example(1)]
    return ",".join(vals)
