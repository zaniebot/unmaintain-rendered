More deprecated aliases of built-in exceptions

#17935 opened by @DimitriPapadopoulos on 2025-05-08 06:24
@DimitriPapadopoulos
Summary

Update os-error-alias (UP024) for the first one:

  • resource.error is a deprecated alias of OSError:

    Changed in version 3.3: Following PEP 3151, this class was made an alias of OSError.

  • ExecError will be a deprecated alias of RuntimeError, starting with Python 3.14:

    The ExecError exception has been deprecated since Python 3.14. It has not been used by any function in shutil since Python 3.4, and is now an alias of RuntimeError.
