Mark redefined variables as unused in simple same-branch cases

Issue opened by @charliermarsh on 2022-12-10 04:37

Given this code:

def f():
    x = 1
    x = 2

We should be able to mark x = 1 as redefined-while-unused.

Because we don't do full branch analysis, I don't think we can flag cases like:

def f():
    x = 1
    if True:
      x = 2
