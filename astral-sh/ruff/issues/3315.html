class Dummy:
    """Dummy class for testing."""

    def __init__(self, arg1, arg2, arg3):
        """
        Args:
            arg1: some arg
            arg2: some arg
            arg3: some arg
        """

to


class Dummy:
    """Dummy class for testing."""

    def __init__(self, arg1, arg2, arg3):
        """Args:
        arg1: some arg
        arg2: some arg
        arg3: some arg.
        """
        pass

The de-indenting of args in particular is bad. Could we give 'Args': special treatment and always leave it on its own line?
