# disable formatting lints (a single option)

---

_Issue opened by @KDruzhkin on 2024-03-04 09:24_

I usually enable all stable lints, and then disable some:

```toml
[tool.ruff.lint]
preview = false
select = ["ALL"]
ignore = [
   "WHAT", # comment why
   "WHAT", # comment why
   "WHAT", # comment why
]
```

Now the linter forces me to add 15 exceptions to the 'ignore' list. The ruff linter documentation says: _When using Ruff as a formatter, we recommend avoiding the following lint rules: ..._ (<https://docs.astral.sh/ruff/formatter/#conflicting-lint-rules>)

**Can we have a single lint config option to switch off formatting lints?**

```toml
[tool.ruff.lint]
preview = false
format = false  # â˜œ
```
