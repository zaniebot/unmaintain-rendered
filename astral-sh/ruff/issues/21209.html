RUF057 has false positives with starred arguments

#21209 opened by @dscorbett on 2025-11-02 22:28
Issue opened by @dscorbett on 2025-11-02 22:28

Summary

unnecessary-round (RUF057) has false positives with starred arguments. Example:

$ cat >ruf057.py <<'# EOF'
print(round(125, **{"ndigits": -2}))
print(round(125, *[-2]))
# EOF

$ python ruf057.py
100
100

$ ruff --isolated check ruf057.py --select RUF057 --fix
Found 2 errors (2 fixed, 0 remaining).

$ cat ruf057.py
print((125))
print((125))

$ python ruf057.py
125
125

Version

ruff 0.14.3 (8737a2d5f 2025-10-30)
