e.g. in

class C1: pass
class C2: pass
class C3: pass
x = C1
if flag1:
    x = C2
elif flag2:
    x = C3

we should display the type as (Literal[C1] | Literal[C2] | Literal[C3]), without extraneous parens (representing extraneous nested unions we didn't flatten.)
