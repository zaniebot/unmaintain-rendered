Summary

Hi, ICN003 says this is good and this is bad:

  • Good
    • import pandas as pd
  • Bad
    • from pandas import ...

Can you add a rule to check the opposite?

Example:

  • Good
    • from typing import Callable, Any
  • Bad
    • import typing and then typing.Callable
