Comment by @mehalter on 2024-08-13 14:48

This was resolved by correctly adding the code from the ruff documentation.

No issue here on the ruff side ðŸ˜„

return {
  "AstroNvim/astrolsp",
  opts = {
    config = {
      ruff = {
        init_options = {
          settings = {
            lint = { enable = false },
          },
        },
      },
    },
  },
}
