UP019 should detect typing_extensions.Text

#20819 opened by @dscorbett on 2025-10-12 14:29
Issue opened by @dscorbett on 2025-10-12 14:29

Summary

typing-text-str-alias (UP019) detects typing.Text. It would also be useful for it to detect typing_extensions.Text. Example:

$ cat >up019.py <<'# EOF'
from typing_extensions import Text
foo: Text = "bar"
# EOF

$ ruff --isolated check up019.py --select UP019
All checks passed!
