Comment by @zanieb on 2023-09-14 13:55

Hi! These rules are in preview. See our documentation for details.

Using the --preview flag will select those rules.

There's a bug in our documentation where the preview indicator is missing. It'll be restored in our next release (fixed in astral-sh/ruff#7341)
