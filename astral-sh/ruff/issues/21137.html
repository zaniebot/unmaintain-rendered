Hi everyone, I was wondering if you have already considered (or would consider) adding a "ruff init" command, similar to cz init or eslint for instance. Goal is to provide a CLI tool that allows user to build their config in a convenient way by answering a few prompts.

This looks like something fun to work on, so I'd be happy to do it. On the other hand, maybe it's not the priority. But what are your thoughts on this?
