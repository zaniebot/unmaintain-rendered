Comment by @Jackenmen on 2023-03-02 08:32

The error is that you're supposed to use constant strings, not dynamic ones as that passed string is used as a key during the lookup in message catalog.

I think that ideally this should support not just the most basic gettext call accepting a single argument but also calls to gettext, ngettext, and so on: https://docs.python.org/3/library/gettext.html
