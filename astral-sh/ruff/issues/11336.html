Hi @darowny

No, there's no such formatter setting.

The hugging preview style only applies if the array (or dict or set) is the only argument in a function call or the only content of a prenthesized expression

trans = call([
    {
        "first": Stuff,
        "second": Other,
        "third": Blah,
        "third": Blah,
    }
])


trans = a + ([
    {
        "first": Stuff,
        "second": Other,
        "third": Blah,
        "third": Blah,
    }
])
