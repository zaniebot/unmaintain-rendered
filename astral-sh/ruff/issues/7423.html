Formatter incompatibility: extra parentheses around unary expression

Closed #7423 opened by @charliermarsh on 2023-09-15 21:15
Issue opened by @charliermarsh on 2023-09-15 21:15

Given:

if True:
    if True:
        if True:
            if not yn_question(
                Fore.RED
                + "WARNING: Removing listed files. Do you really want to continue. yes/n)? "
            ):
                sys.exit(1)

Ruff formats as:

if True:
    if True:
        if True:
            if (
                not yn_question(
                    Fore.RED
                    + "WARNING: Removing listed files. Do you really want to continue. yes/n)? "
                )
            ):
                sys.exit(1)

Black formats as:

if True:
    if True:
        if True:
            if not yn_question(
                Fore.RED
                + "WARNING: Removing listed files. Do you really want to continue. yes/n)? "
            ):
                sys.exit(1)
