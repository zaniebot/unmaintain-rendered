Update `UP032` to handle repeated indices/keywords

Closed #6217 opened by @harupy on 2023-08-01 01:42
Issue opened by @harupy on 2023-08-01 01:42

Example:

x = 1
print("{0} and {0}".format(x))
print("{a} and {a}".format(a=x))

Expected behavior:

UP032 is raised and the code is fixed to:

x = 1
print(f"{x} and {x}")
print(f"{x} and {x}")

Actual behavior:

UP032 is not raised (playground)
