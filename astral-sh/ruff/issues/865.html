Something like that! It doesn't officially stand for anything, but I did want something that somehow incorporated Rust and Flake phonetically. I also liked that Ruff is: (1) very easy to type (helpful for a command-line tool); and (2) a homophone of "rough", since I released it in a very rough state to start.
