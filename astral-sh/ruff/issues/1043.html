Closed #1043 opened by @charliermarsh on 2022-12-04 19:28

import os
import os as os

Ruff reports an unused import on line 2, although that's an explicit re-export:

It's easy to fix:

import os
import os as os

But harder to fix:

import os as os
import os

Fixing the latter requires that we can detect redefinition (see: #996).
