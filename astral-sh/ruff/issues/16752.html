F841 does not have branch analysis

Open #16752 opened by @sk- on 2025-03-14 15:46
Issue opened by @sk- on 2025-03-14 15:46

Summary

F841 does not report variables that are unused in certain branches. Below is a short example (playground) showing the issue

def func(x):
    if x:
        io = ""
    else:
        io = "foo"
        print(io)

This issue was originally reported to pyflakes and closed as duplicate of https://github.com/PyCQA/pyflakes/issues/715.

They mention is unlikely they will implement branch analysis, but ruff already has (I guess) the machinery needed to support this use case.

Version
