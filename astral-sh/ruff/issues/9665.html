CPY001 with multiple authors

#9665 opened by @jonyscathe on 2024-01-29 05:20
@jonyscathe

No obvious way to have multiple copyright authors.
It is quite plausible that one module would require a different copyright author in different files. In flake8-copyright this is possible with the setting: copyright-author = 'Author1|Author2' as the copyright-author field is regex. So that way if either Author1 or Author2 is in the file's copyright string there is no error. Is there currently any way to get this working with ruff?
