Comment by @charliermarsh on 2023-03-22 20:01

Unfortunately I don't know that we have a great tutorial beyond the tutorial in the docs, which is mostly focused on command-line usage. But I'm happy to answer questions here or in Discord.

In this case, I think what you're experiencing is that if you use ruff.toml (instead of pyproject.toml), you should omit the [tool.ruff], so your entire file would be:

ignore = ["E501"]

So, Ruff is probably trying to read that file and failing, and the error is being hidden in the VS Code logs.
