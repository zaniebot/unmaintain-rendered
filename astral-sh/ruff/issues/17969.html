Inconsistent formatting of match-case on `[]` and `_`

Closed #17969 opened by @adamchainz on 2025-05-08 21:37
Issue opened by @adamchainz on 2025-05-08 21:37

Summary

Playground link

Source:

a = []
b = []
match a, b:
    case [], []:
        ...
    case [], _:
        ...
    case _, []:
        ...
    case _, _:
        ...

Formatted:

a = []
b = []
match a, b:
    case [[], []]:
        ...
    case [[], _]:
        ...
    case _, []:
        ...
    case _, _:
        ...

It seems rather inconsistent that the formatter adds wrapping [] on the first two cases but not the others.
