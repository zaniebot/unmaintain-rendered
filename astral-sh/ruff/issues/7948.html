Insert newline after docstring even when there are comments

Closed #7948 opened by @konstin on 2023-10-13 15:50
Issue opened by @konstin on 2023-10-13 15:50

Ours:

class ModuleBrowser:
    """Browse module classes and functions in IDLE."""
    # This class is also the base class for pathbrowser.PathBrowser.

    def __init__(self, master, path, *, _htest=False, _utest=False):
        pass

Black:

class ModuleBrowser:
    """Browse module classes and functions in IDLE."""

    # This class is also the base class for pathbrowser.PathBrowser.

    def __init__(self, master, path, *, _htest=False, _utest=False):
        pass

We should also insert the empty line after the docstring even if there is comment
