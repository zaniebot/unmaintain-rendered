False-positive for D208 "Docstring is over-indented" for code sample at the end of module docstring

#986 opened by @tmke8 on 2022-12-01 18:37
Issue opened by @tmke8 on 2022-12-01 18:37

I don't know if this appears anywhere else, but I observed it in a module docstring.

This:

"""Module docstring.

.. code:: python
   a = f()
   assert a
"""

gives the error:

Found 1 error(s).
main.py:5:1: D208 Docstring is over-indented

whereas this:

"""Module docstring.

.. code:: python
   a = f()
   assert a

More documentation.
"""

gives no error.

I can understand though if this is a wont-fix because I can just turn off the D208 error code.
