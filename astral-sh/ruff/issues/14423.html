Wrong fix for PLC2801

#14423 opened by @bebound on 2024-11-18 03:50
@bebound

ruff check kk.py --select PLC2801 --fix --preview --unsafe-fixes --isolated change the code from

if 'kk'.__contains__("k"):
    print("yes")

to

if 'kk' in "k":
    print("yes")

The correct fix is if 'k' in 'kk'.

Version: ruff 0.7.4
