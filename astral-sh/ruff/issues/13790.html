new rule - replace if all([value_1, value_2]) with if value_1 and value 2

#13790 opened by @spaceby on 2024-10-17 09:04
@spaceby

In a project I'm working on, I saw this code

value_1 = value_2 = value_3 = True

if all([value_1, value_2, value_3]):
    pass

t can be rewritten like this

value_1 = value_2 = value_3 = True

if value_1 and value_2 and value_3:
    pass

This makes the code simpler and faster. The same logic can be applied to any, tuple
