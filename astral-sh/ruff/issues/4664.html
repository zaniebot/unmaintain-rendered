#4664 opened by @JannKleen on 2023-05-26 08:21

When having the following code:

def test():  # noqa: too-many-branches
  pass

RUF100 would only remove the noqa, leaving this:

def test():  # : too-many-branches
  pass

Edit: The removal of the 'numeric' codes (i.e. # noqa: PLR0912) works as expected.
