Config option `classes` from isort (and `constants` and `variables` probably)

Closed #1819 opened by @tmke8 on 2023-01-12 15:58
Issue opened by @tmke8 on 2023-01-12 15:58

The order in isort is CONSTANTS, Classes, functions, which is detected by the case of the letters, but sometimes classes are in all-caps, like SVC from sklearn or SELU from pytorch.

In order to sort them correctly, I can specify classes:

[tool.isort]
classes = ["SVC", "SELU"]

and then isort treats them like classes even though they're in constant-case.

  • [ ] variables
  • [x] classes
  • [ ] constants
