Don't move type param opening parenthesis comments

#8162 opened by @dhruvmanila on 2023-10-24 11:29
@dhruvmanila

Given:

type foo[  # comment
    a,
    b
] = ...

We format it as:

type foo[a, b] = ...  # comment

But, the list formatting is from:

foo = [  # comment 0
    a,
    b
]

To:

foo = [  # comment 0
    a,
    b,
]
