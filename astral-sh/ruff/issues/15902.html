Allow common numbers for PLR2004

#15902 opened by @spacemanspiff2007 on 2025-02-03 05:50

Description

When dealing with bytes it's common do divide by 1024 to get KiByte, MiByte, etc.. Thus this should not trigger PLR2004:

b: bytes = b'asdf'
if len(b) > 1024:
    print('More than 1 KiB')
