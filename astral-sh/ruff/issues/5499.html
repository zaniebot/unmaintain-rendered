Support `FORCE_COLOR` env var

#5499 opened by @MicaelJarniac on 2023-07-04 04:07
Issue opened by @MicaelJarniac on 2023-07-04 04:07

Feature Request

Add support for the FORCE_COLOR environment variable. When set, regardless of value, all output should have colors enabled, as if the terminal supported them.

As far as I can tell, Ruff does support a similar environment variable, CLICOLOR_FORCE. It seems to me, though, that FORCE_COLOR is a lot more common. In my opinion, it'd be worth supporting both.

Some projects that use this:

  • Pytest
  • Nox
  • Tox
  • Rich
  • Mypy
  • build
