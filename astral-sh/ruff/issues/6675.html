Formatter: Optional empty line after class docstring

Closed #6675 opened by @MichaReiser on 2023-08-18 13:25
Issue opened by @MichaReiser on 2023-08-18 13:25

This comes up frequently in the django project.

Input

class UnaryPredicate(GEOSFuncFactory):
    "For GEOS unary predicate functions."
    argtypes = [GEOM_PTR]

class UnaryPredicate2(GEOSFuncFactory):
    "For GEOS unary predicate functions."

    argtypes = [GEOM_PTR]

Black Black preserves up to 1 empty line after the class docstring but doesn't insert one if it is missing.

class UnaryPredicate(GEOSFuncFactory):
    "For GEOS unary predicate functions."
    argtypes = [GEOM_PTR]

class UnaryPredicate2(GEOSFuncFactory):
    "For GEOS unary predicate functions."
