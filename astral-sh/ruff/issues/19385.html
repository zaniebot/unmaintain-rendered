Summary

The fix for ellipsis-in-non-empty-class-body (PYI013) can remove comments, which is unnecessary and unsafe. Example:

$ cat >pyi013.py <<'# EOF'
class C:
    x = 1
    ...  # comment
# EOF

$ ruff --isolated check pyi013.py --select PYI013 --diff
--- pyi013.py
+++ pyi013.py
@@ -1,3 +1,2 @@
 class C:
     x = 1
-    ...  # comment
