Thanks -- thanks has been requested before and can be tracked here: astral-sh/ruff#1203. Right now, though, we only respect # noqa directives. You can run ruff check --select RUF100 --fix to automatically add # noqa directives for existing violations, if you want, or even ruff check --select RUF100 --select F401 --fix to add # noqa for existing unused imports.
