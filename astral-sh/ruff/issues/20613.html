Sorry, I got confused by your other issue that was about add-noqa. You can run ruff check --fix src and it should remove unused noqa comments. You don't need to select any additional rules if RUF100 is already part of your normal rule set.

You can use fixable and unfixable if you only want to apply the RUF100 fixes:

ruff check --fixable RUF100 --unfixable ALL
