Autofix for ANN 201 incorrectly adds `None` Return Type

Closed #9201 opened by @max-muoto on 2023-12-19 17:57
Issue opened by @max-muoto on 2023-12-19 17:57

Let's say we have a function that has no ability to return:

def foo():
    raise ValueError

The correct return type for this would be NoReturn. However, the autofix for ANN 201, adds None here as the return type instead.

Ruff Version: 0.1.8 Command: ruff check . --fix --unsafe-fixes.
