The first parenthesis is dropped in the autofix here and the second parenthesis ends up in he wrong place.

+++ test.py
@@ -1,3 +1,3 @@
 number = 1
-if 0 < (number - 100):

It should be like this:

--- test.py
+++ test.py
@@ -1,3 +1,3 @@
 number = 1
-if 0 < (number - 100):
+if (number - 100) > 0:
     print(number)
