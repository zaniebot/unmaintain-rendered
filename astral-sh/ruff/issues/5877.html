error: Failed to initialize cache at /app/.ruff_cache: Permission denied (os error 13)
error: Failed to create cache file '/app/.ruff_cache/content/14915633640191433684'

ruff check .

Version is 0.0.272

Is there a possibility to run ruff without a cache?
