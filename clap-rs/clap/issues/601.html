Docs incorrect about ArgGroups in YAML

#601 opened by @laishulu on 2016-07-25 06:05
@laishulu

with the following yaml file, when I invoke myapp without any arguments, it just passes. But according to the spec, at least one of base and delta should be presented. Also, I don't find anything about the arg group when I invoke myapp help

name: myapp
version: 0.1
about: myapp
global_settings:
    - ColoredHelp
args:
    - base:
        long: base
        value_name: BASE
        help: Base
        takes_value: true
        display_order: 2
    - delta:
        long: delta
        value_name: DELTA
        help: delta
        takes_value: true
        display_order: 3
arg_groups:
    - test:
        args:
            - base
            - delta
        required: true
