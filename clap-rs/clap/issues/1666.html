Version does not exist? How do I install version 3?

Comment by @CreepySkeleton on 2020-02-04 07:07

3.x version was not released yet, officially. We will release 3.0.0-alpha.1 as soon as it's done, I expect it to be a week or two from now on.

Meanwhile, you can try the upcoming version via

clap = { git = "https://github.com/clap-rs/clap" }

Please be aware that clap is under heavy development - any help is very welcome here! - so expect severe bugs, documentation inconsistency and API breakage.
