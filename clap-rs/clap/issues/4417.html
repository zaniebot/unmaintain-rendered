Comment by @epage on 2022-10-24 13:10

The error message is better under debug builds than release builds.

This error is expected behavior and I would be more surprised by this working than not. ArgAction::Count has only worked with u8's. The old parse(from_occurrences) did support alternative data types. The fix is to either change the data type to u8 or to use parse(from_occurrences) though that will be unavailable in v4. clap-rs/clap#3912 is one potential way of allowing alternative data types.

As this error is expected, I'm going to go ahead and close this. If there is something I was missing in closing this, let us know!.
