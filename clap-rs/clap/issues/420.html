Positional Args with Value Names prints Arg Name

#420 opened by @kbknapp on 2016-02-08 21:22
@kbknapp
extern crate clap;

use clap::{App, Arg};

fn main() {
    let m = App::new("valnames")
        .arg(Arg::with_name("arg")
            .required(true)
            .value_name("FILE"))
        .get_matches();
}

Help prints:

$ valnames --help
valnames 

USAGE:
    valnames [FLAGS] [ARGS] <arg>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

ARGS:
    arg

Note arg
