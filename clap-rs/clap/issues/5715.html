Comment by @epage on 2024-09-03 14:42

tl;dr use get_flag instead of contains_id.

            println!("aa: {}", arg.contains_id("aa"));
            println!("bb: {}", arg.contains_id("bb"));

Might not be fully obvious but from the contains_id docs:

NOTE: This will always return true if default_value has been set. ArgMatches::value_source can be used to check if a value is present at runtime.

If you look at the SetTrue docs, it calls out that a default_value is set automatically and the examples show that contains_id is always true while get_flag changes based on whether the flag is present or not.
